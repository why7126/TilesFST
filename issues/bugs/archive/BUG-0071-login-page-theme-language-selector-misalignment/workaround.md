# Workaround - BUG-0071 登录页右上角主题选择模块与语言选择模块没有对齐

## 临时规避方案

暂无需要对用户发布的临时规避方案。

在修复前，用户仍可正常使用以下能力：

- 使用账号密码登录管理端。
- 使用主题选择模块切换界面主题。
- 点击语言选择模块。

该缺陷属于登录页视觉对齐问题，不需要后台数据修复、接口回滚、配置调整或运维干预。

## 运维 / 客服说明

如用户反馈登录页右上角主题选择与语言选择未对齐，可说明：

1. 当前登录、主题切换和语言按钮展示能力可用。
2. 问题属于登录页工具区视觉布局偏差。
3. 后续修复会统一右上角工具区的控件对齐和响应式表现。

## 风险

临时不处理不会导致数据损坏、登录失败或权限风险，但会持续影响 Web 管理端登录页首屏视觉一致性。
