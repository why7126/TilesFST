# 回归验收标准

## 页面范围

以下页面 MUST 纳入同一轮验收矩阵：

- [ ] 瓷砖 SKU
- [ ] 瓷砖品牌
- [ ] 瓷砖类目
- [ ] 瓷砖规格
- [ ] Banner 管理
- [ ] 用户管理
- [ ] 日志审计
- [ ] 接口文档

## 布局验收

- [ ] **AC-001** 上述页面模块顺序 MUST 统一为「标题模块 → 指标卡模块 → 筛选/搜索模块 → 列表模块」。
- [ ] **AC-002** 标题模块 MUST 保留页面标题、眉标、说明文案和主操作入口，且不被筛选区或指标卡打断。
- [ ] **AC-003** 指标卡模块 MUST 位于筛选/搜索模块之前，Banner 管理等当前顺序不一致页面 MUST 调整。
- [ ] **AC-004** 列表模块 MUST 位于筛选/搜索模块之后，表格上方不得新增割裂的重复标题或旧版 toolbar 文案。

## 筛选/搜索验收

- [ ] **AC-005** 筛选/搜索模块交互和视觉 MUST 以瓷砖 SKU 页为基线，并形成跨页面一致的字段排列、按钮对齐和控件高度。
- [ ] **AC-006** 所有页面 MUST 移除【查询】按钮，不得出现「查询」「搜索」等显式提交筛选按钮，除非后续评审明确豁免。
- [ ] **AC-007** 筛选条件变化后 MUST 自动重置页码为第 1 页，并刷新或重新计算列表结果。
- [ ] **AC-008** 所有页面 MUST 保留【重置】按钮，重置按钮尺寸、边框、圆角、字号、图标策略和对齐方式 MUST 保持一致。
- [ ] **AC-009** 点击【重置】后 MUST 清空或恢复默认筛选条件，并将页码重置为第 1 页。
- [ ] **AC-010** 筛选/搜索模块 MUST 使用 Design System token 和既有 `btn`、`input`、`select` 等基础样式，不得新增裸 Hex 或割裂的局部按钮样式。

## 表格与固定操作列验收

- [ ] **AC-011** 上述页面列表最后一列 MUST 统一为固定浮动操作列，视觉和交互以接口文档页为基线。
- [ ] **AC-012** 固定操作列 MUST 在横向滚动时保持可见，并提供与接口文档页一致的右侧背景、左侧分割线和阴影层次。
- [ ] **AC-013** 表头最后一列与表体最后一列固定行为 MUST 一致，不得出现表头/表体错位。
- [ ] **AC-014** 行 hover 时固定操作列背景 MUST 与当前行 hover 状态协调，不得出现断层或透明露底。
- [ ] **AC-015** 固定操作列中的编辑、启停、删除、查看、重置密码等操作 MUST 保持现有权限、禁用态和二次确认逻辑不回退。

## 分页验收

- [ ] **AC-016** 所有页面分页区域 MUST 使用统一 DOM 结构：左侧 `page-summary`，右侧 `page-right`。
- [ ] **AC-017** 分页按钮 MUST 使用统一 `page-buttons`、`page-btn`、`active` 样式。
- [ ] **AC-018** 分页最多展示 5 个可点击页码，不包含上一页/下一页按钮。
- [ ] **AC-019** 当总页数大于 5 时，页码窗口 MUST 围绕当前页滚动，且不得超过 5 个可点击页码。
- [ ] **AC-020** 当总页数为 1 时，分页控件 MUST 保持统一结构，上一页/下一页为禁用态，页码 `1` 为当前态。
- [ ] **AC-021** 切换每页显示条数 MUST 将页码重置为第 1 页。
- [ ] **AC-022** 分页总数 MUST 基于当前筛选结果计算。

## 回归验收

- [ ] **AC-023** 各页面原有新增、编辑、启停、删除、查看、重置密码等主流程不回退。
- [ ] **AC-024** 各页面权限边界不回退，管理端仍仅允许已授权用户访问。
- [ ] **AC-025** 修复不修改后端 API 契约、不修改数据库结构、不修改 MinIO 或媒体上传策略。
- [ ] **AC-026** 若修复仅涉及前端布局、筛选触发和分页呈现，不需要执行 Orval；若改动接口分页结构，则 MUST 同步 OpenAPI 与 Orval。
- [ ] **AC-027** Web 管理端在 1366px、1440px、1920px 常见桌面宽度下，筛选区、表格和分页不得出现明显重叠、裁切或错位。

## 测试建议

- [ ] **AC-028** 为受影响页面补充或更新 Vitest/Testing Library 测试，覆盖模块顺序：标题、指标卡、筛选/搜索、列表。
- [ ] **AC-029** 测试所有受影响页面不再渲染【查询】按钮，并保留统一【重置】按钮。
- [ ] **AC-030** 测试分页最多渲染 5 个可点击页码，并覆盖总页数为 1、5、6 及当前页居中的场景。
- [ ] **AC-031** 测试筛选条件变化后页码重置为 1。
- [ ] **AC-032** 测试每页显示条数变化后页码重置为 1。
- [ ] **AC-033** 测试固定操作列相关 class 或 DOM 契约在各页面表格最后一列存在。
