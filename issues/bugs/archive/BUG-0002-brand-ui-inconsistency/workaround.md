# 临时规避方案

## 1. 可用性规避

该缺陷不阻断品牌管理功能，当前可继续使用：

1. 品牌列表分页仍可通过上一页、当前页、下一页、每页显示和跳页输入完成翻页。
2. 新增/编辑品牌弹窗仍可通过「品牌Logo」区域选择并上传 Logo 文件。
3. 保存品牌资料不受该 UI 一致性缺陷影响。

## 2. 验收规避

在正式修复前，验收时应明确标注：

- 品牌列表分页与用户管理页分页存在视觉差异，暂不作为通过项。
- 品牌 Logo 选择文件控件存在视觉风格差异，暂不作为通过项。
- 仅验证品牌创建、编辑、Logo 上传、分页功能本身是否可用。

## 3. 运营规避

内部管理员可继续按现有路径维护品牌：

1. 进入「瓷砖品牌」。
2. 使用现有分页控件定位品牌。
3. 使用「新增品牌」或「编辑」维护 Logo 和品牌资料。

## 4. 风险说明

该规避方案只能保证功能可用，不能消除以下问题：

- 管理端页面视觉一致性不足。
- 用户管理页与品牌管理页同类控件体验不一致。
- 后续新增管理端列表页可能继续复制不一致的分页结构。

因此仍建议进入 `/bug-review`，评审通过后通过 `fix-*` OpenSpec Change 修复。
