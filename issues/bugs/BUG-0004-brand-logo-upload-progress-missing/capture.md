# 现象

编辑品牌弹窗中，点击「更换 Logo」并选择图片后，文件选择窗口关闭，但没有明显触发图片上传；Logo 预览也没有更新。用户无法判断图片是否正在上传、上传是否成功或需要等待多久。

# 复现步骤

1. 进入 Web 管理端品牌管理页。
2. 打开已有品牌的编辑弹窗。
3. 点击「更换 Logo」。
4. 在系统文件选择窗口中选择一张图片并打开。
5. 观察弹窗中的 Logo 预览和上传反馈。

# 期望 vs 实际

期望：

- 选择图片后应立即触发 Logo 上传。
- 上传过程中应展示进度条或等价进度反馈，让用户感知等待时长。
- 上传成功后，弹窗中的 Logo 预览应更新为新图片。
- 上传失败时应展示明确错误信息，并保留可重试入口。

实际：

- 选择图片后没有明显上传反馈。
- Logo 预览没有更新。
- 缺少上传进度条，用户无法感知上传等待过程。
