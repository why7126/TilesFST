# 用户故事

## US-001 装修业主快速理解品牌与选砖入口

作为装修业主，我希望打开小程序首页后能立即看到菲尚特品牌、搜索入口、主推 Banner 和选砖入口，以便快速判断这里是否能帮我找到合适瓷砖。

验收要点：

- 首页首屏展示品牌 Header、搜索入口、Banner 和四个快捷入口。
- 页面使用深色品牌视觉，品牌金只用于强调、价格、选中态和主要行动。
- 不出现后台管理、库存、订单、编辑等内部能力入口。

## US-002 设计师快速进入筛选与榜单

作为设计师，我希望首页入口能清楚区分选瓷砖、品牌馆、新品榜和热销榜，以便根据客户沟通场景快速进入对应浏览路径。

验收要点：

- 快捷入口固定为选瓷砖、品牌馆、新品榜、热销榜。
- 四个入口采用统一的图标在上、文案在下结构。
- 新品榜和热销榜可复用已有搜索页 section 参数或安全降级，不出现白屏。

## US-003 门店客户连续浏览全部产品

作为门店客户，我希望看完新品和热销后还能继续在首页浏览更多产品，而不是必须先跳转到列表页。

验收要点：

- 热销推荐下方展示“全部产品”双列瀑布流。
- 首次加载展示一批公开 SKU，触底后追加下一页。
- 加载中、加载失败、无更多数据均有明确状态。

## US-004 门店销售展示商品信息

作为门店销售，我希望首页商品卡片展示清晰的主图、SKU、规格和价格，方便面对面向客户介绍产品。

验收要点：

- 新品、热销、全部产品卡片均使用 SKU 主图。
- 商品价格展示使用 `price_display`，不得回退到“到店咨询”等旧文案。
- 商品卡片点击进入商品详情页。

## US-005 用户尝试收藏或证书入口时安全降级

作为用户，我可能会点击原型中的收藏或证书入口。我希望即使相关完整能力还未建设，也不会遇到白屏、报错或无法返回。

验收要点：

- 收藏心形如出现，仅作为视觉反馈或建设中提示，不承诺持久化。
- 收藏 Tab、证书 Tab 如未具备完整页面，必须展示建设中提示或跳转占位页。
- 点击事件可记录埋点，供后续判断是否拆分建设完整能力。
