# 业务流程

## 1. 总流程

```text
用户浏览商品 / 品牌 / 分类
  |
  +-- 点击收藏 ----------------+
  |                            |
  +-- 已存在收藏记录 ----------+
                               |
                               v
                       收藏数据写入或保持
                               |
                               v
                  用户从入口进入收藏列表页
                               |
              +----------------+----------------+
              |                                 |
              v                                 v
          有收藏内容                         无收藏 / 未登录 / 异常
              |                                 |
              v                                 v
       展示收藏列表项                   空状态 / 登录引导 / 重试
              |
              +-- 点击收藏项 --> 对应详情页
              |
              +-- 取消收藏 --> 更新列表并反馈
              |
              +-- 刷新 / 加载更多 --> 同步最新收藏状态
```

## 2. 进入收藏列表流程

| 步骤 | 说明 | 输出 |
|---|---|---|
| 1 | 用户点击收藏列表入口 | 进入收藏列表页 |
| 2 | 系统识别登录态或访客标识 | 决定请求用户收藏或展示登录引导 |
| 3 | 请求第一页收藏数据 | 展示加载态或骨架屏 |
| 4 | 返回收藏内容 | 渲染收藏列表项 |
| 5 | 无收藏内容 | 展示空状态和浏览引导 |
| 6 | 网络失败 | 展示错误状态和重试入口 |

## 3. 收藏项浏览与跳转流程

| 收藏对象 | 列表展示 | 点击后 |
|---|---|---|
| SKU / 商品 | 主图、名称、规格、品牌、参考价格或摘要 | 进入 SKU / 商品详情页 |
| 品牌 | 品牌 Logo 或占位、品牌名、简介摘要 | 进入品牌详情页或品牌商品列表 |
| 分类 | 分类名、层级、示意图或占位 | 进入分类商品列表页 |

首期范围若仅支持 SKU / 商品，品牌和分类流程标记为后续扩展，不进入 v1 验收。

## 4. 取消收藏流程

```text
用户点击取消收藏
  |
  v
展示处理中状态
  |
  +-- 成功 --> 从当前列表移除或灰态撤销窗口 --> 更新收藏数量
  |
  +-- 失败 --> 保留原列表项 --> 展示重试提示
```

规则：

- 取消收藏不得误触发进入详情。
- 取消收藏失败不得把本地 UI 误更新为已取消。
- 是否提供撤销窗口待后续产品确认；如不提供，必须有明确成功反馈。

## 5. 状态流程

| 场景 | 处理 |
|---|---|
| 首屏加载 | 展示列表骨架或等价加载态 |
| 有收藏 | 展示收藏列表，支持点击详情与取消收藏 |
| 无收藏 | 展示空状态，并引导去首页、分类或搜索 |
| 未登录 | 展示登录引导或访客态说明 |
| 对象下架/删除 | 灰态展示、提示不可查看或按产品策略隐藏 |
| 首屏失败 | 展示错误状态和重试 |
| 加载更多失败 | 保留已加载内容并提供重试 |

## 6. 与关联需求差异

| 关联需求 | 差异 |
|---|---|
| REQ-0047-product-list-common-component-application | 商品列表负责发现和筛选商品；收藏列表负责回访已保存对象 |
| REQ-0044-miniapp-sku-detail-page | SKU 详情页是收藏项跳转目标之一，本需求不改变详情页字段 |
| REQ-0045-category-list-page | 分类页用于发现商品，收藏列表空状态可回流分类页 |
| REQ-0046-search-component-application | 搜索用于主动查找，收藏列表空状态可回流搜索 |

## 7. 非目标流程

- 不进入管理端收藏运营、批量导出或人工维护流程。
- 不处理购物车、询价单、在线下单或客服会话。
- 不提供收藏分组、分享收藏夹或多人协作。
- 不把浏览历史、足迹或推荐结果混入收藏列表。
