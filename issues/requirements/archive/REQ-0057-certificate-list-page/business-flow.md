# 业务流程

## 1. 总览

```text
用户进入小程序
  |
  +-- 点击 TabBar「证书」
        |
        +-- 进入 pages/certificates/index
              |
              +-- 加载公开证书列表 API
                    |
                    +-- 成功且有数据 -> 展示证书卡片列表
                    |       |
                    |       +-- 搜索 / 筛选 -> 重置页码并重新加载
                    |       +-- 下拉刷新 -> 刷新第一页
                    |       +-- 触底 -> 加载更多
                    |       +-- 点击证书 -> 图片预览 / PDF 受控打开
                    |
                    +-- 成功但无数据 -> 展示空状态
                    |
                    +-- 失败 -> 展示错误态与重试
```

## 2. 数据发布流程

```text
管理端维护品牌证书（REQ-0038）
  |
  +-- 上传证书文件、填写证书元数据
  |
  +-- 设置 is_visible=true
  |
  +-- 后端公开证书列表 API 过滤
        |
        +-- 未删除
        +-- 前台展示
        +-- 所属品牌允许公开
        +-- 文件 URL 为受控读取地址
  |
  +-- 小程序证书 Tab 展示
```

## 3. 与父 REQ 差异

| 维度 | REQ-0038 品牌证书管理 | REQ-0057 证书列表页 |
|---|---|---|
| 终端 | Web 管理端 | 微信小程序 |
| 用户 | 企业内部运营/管理员 | 装修客户、设计师、门店访客 |
| 能力 | 创建、编辑、上传、显示/隐藏、删除、审计 | 公开浏览、搜索筛选、预览 |
| 权限 | 管理端认证与权限点 | 公开只读，不暴露内部字段 |
| 布局 | 管理端表格 + 筛选 + 弹窗 | 移动端单列卡片 + TabBar |
| 文件处理 | 上传与管理预览 | 公开预览或受控打开 |

## 4. 异常流程

### 4.1 证书无公开数据

```text
API 返回 items=[]
  |
  +-- 无筛选条件 -> 展示「暂无公开证书」
  +-- 有筛选条件 -> 展示「没有符合筛选条件的证书」+ 清除筛选入口
```

### 4.2 文件预览失败

```text
用户点击证书文件
  |
  +-- file_url 缺失 -> 展示「证书文件暂时无法预览」
  +-- 图片预览失败 -> 展示失败提示，可返回列表
  +-- PDF 打开失败 -> 展示失败提示或复制受控链接入口
```

### 4.3 分页重复触发

```text
触底加载更多
  |
  +-- loadingMore=true -> 忽略重复触发
  +-- hasMore=false -> 展示无更多提示
  +-- 请求失败 -> 保留已加载列表并允许重试
```
