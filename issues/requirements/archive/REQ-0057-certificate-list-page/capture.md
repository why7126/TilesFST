# 新增证书列表页

面向证书展示/管理场景，新增证书列表页，用于集中浏览品牌或产品相关证书信息，并承接后续查看、筛选或管理动作。

# 原始描述

新增证书列表页

# 待澄清

- [ ] 证书列表页的使用端：Web 展示端、微信小程序、管理端，或多端均需要。
- [ ] 列表项展示字段：证书名称、证书图片、证书类型、品牌、有效期、状态、排序等是否需要。
- [ ] 是否需要搜索、筛选、分页、空状态、加载态和错误态。
- [ ] 是否需要从品牌详情、证书管理页或首页入口跳转进入。
- [ ] 是否只展示已发布证书，还是需要管理端展示草稿/下架等内部状态。
