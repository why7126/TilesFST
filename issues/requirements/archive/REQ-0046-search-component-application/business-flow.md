# 业务流程

## 1. 总流程

```text
用户进入小程序页面
  |
  v
点击搜索入口组件
  |
  v
进入搜索首页
  |
  +-- 点击最近搜索 / 热门搜索 / 最近浏览 ------+
  |                                      |
  v                                      |
输入关键词                              |
  |                                      |
  +-- 未达触发阈值 --> 保持输入态         |
  |                                      |
  +-- 达触发阈值 --> 300ms 防抖请求联想   |
  |                                      |
  v                                      |
展示联想列表                            |
  |                                      |
  +-- 点击可直达实体 --> SKU详情/品牌页/类目页
  |
  +-- 点击关键词建议或提交搜索
       |
       v
   搜索结果页
       |
       +-- 有结果 --> 类型 Tab / 最佳匹配 / 分区结果
       |              |
       |              +-- 应用筛选 --> 刷新结果与筛选数量
       |              |
       |              +-- 点击 SKU --> SKU 详情页
       |
       +-- 无结果 --> 无结果页 / 推荐搜索词
```

## 2. 搜索入口流程

| 步骤 | 说明 | 输出 |
|---|---|---|
| 1 | 页面渲染搜索入口组件 | 展示占位文案和当前页面搜索范围 |
| 2 | 用户点击搜索框 | 进入独立搜索页或当前页搜索态 |
| 3 | 组件传递 `sourcePage` 与 `scope` | 搜索首页、联想和结果页可识别来源 |
| 4 | 用户返回或取消 | 回到原页面，原页面状态不被重置 |

## 3. 联想流程

| 步骤 | 说明 | 规则 |
|---|---|---|
| 1 | 用户输入关键词 | 中文 1 字、英文/数字 2 字后触发 |
| 2 | 触发防抖 | 300ms 内继续输入则重置计时 |
| 3 | 发起联想请求 | 请求携带关键词、搜索范围、limit 和 requestId |
| 4 | 接收响应 | 旧 requestId 的响应不得覆盖新关键词 |
| 5 | 展示联想 | 6 到 10 条；证书不参与联想 |
| 6 | 用户点击建议 | 可直达实体进详情或列表，不可直达建议进结果页 |

## 4. 结果与筛选流程

| 步骤 | 说明 | 规则 |
|---|---|---|
| 1 | 用户提交关键词 | 进入搜索结果页，综合 Tab 默认激活 |
| 2 | 加载结果 | 展示最佳匹配、类型数量、分区和列表 |
| 3 | 用户切换类型 Tab | 展示对应类型结果，保留关键词 |
| 4 | 用户打开筛选抽屉 | 展示品牌、类目、规格、价格区间 |
| 5 | 用户确认筛选 | 刷新结果、聚合数量和当前筛选状态 |
| 6 | 用户重置筛选 | 清空筛选条件，保留关键词 |

## 5. 无结果流程

```text
提交关键词
  |
  v
搜索结果为空
  |
  v
展示关键词 + 无结果说明
  |
  +-- 用户点击推荐搜索词 --> 重新提交搜索
  |
  +-- 用户修改关键词 ------> 回到输入/联想流程
```

## 6. 与关联需求差异

| 关联需求 | 差异 |
|---|---|
| REQ-0041-miniapp-home | 首页只提供搜索入口，本需求承接搜索首页、联想和结果体验 |
| REQ-0044-miniapp-sku-detail-page | SKU 详情页是搜索结果出口，本需求不修改详情页展示规则 |
| REQ-0045-category-list-page | 分类页可携带类目上下文进入搜索，本需求提供搜索组件和结果能力 |

## 7. 非目标流程

- 不进入 Web 管理后台搜索配置中心。
- 不进入店主 Web 搜索流程。
- 不提供询价、购物车、在线下单、客服找砖或联系商家流程。
- 不处理后台热门词、同义词和自然语言词典维护流程。
