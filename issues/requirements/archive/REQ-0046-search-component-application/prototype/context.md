# 微信小程序搜索原型上下文 v1

## 设计基线
与菲尚特小程序首页 v6 保持一致：`#C8A055` 品牌金、`#18160F` 页面背景、`#211E16` 卡片背景、`#EDE8DF` 主文字、半透明辅助文字与分割线。设备画布 390px，圆角 12～16px，搜索框高度 44px。

## 页面清单
1. `01-search-home.html`：历史、热门、最近浏览。
2. `02-search-suggestions.html`：输入态与 6～10 条联想。
3. `03-search-results.html`：最佳匹配、类型 Tab、混合分区。
4. `04-search-filter.html`：快捷筛选与底部完整筛选抽屉。
5. `05-search-empty.html`：无结果状态。

## 交互约束
- 搜索框固定在顶部安全区下方。
- 返回按钮与清空输入按钮均需 44px 可点击区域。
- 搜索结果页类型 Tab 横向可滚动，综合默认激活。
- 筛选已选状态使用品牌金边框与浅金背景。
- SKU 卡片整卡点击进入详情，不显示收藏/分享。
- 证书只在完整结果中出现，不参与联想。
- 底部筛选抽屉覆盖页面约 72%，背景遮罩 62%。

## 数据示例
品牌：蒙娜丽莎、东鹏、马可波罗；类目：岩板、大理石瓷砖；规格：750×1500、900×1800；SKU：FST715001 等。价格统一展示零售价并带单位。
