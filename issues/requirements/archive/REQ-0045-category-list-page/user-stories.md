# 用户故事

## US-001 浏览一级分类

作为装修客户、设计师或门店访客，我希望在分类页左侧看到完整的一级分类导航，以便快速切换到目标瓷砖品类。

验收要点：
- 一级分类仅展示名称，按后台排序升序排列。
- 一级分类超过可视范围时，左栏可独立纵向滚动。
- 当前一级分类通过品牌金指示条、深色选中背景和高亮文字共同表达。
- 切换一级分类时，左栏滚动位置保持不变。

## US-002 浏览二级分类

作为访客，我希望在右侧看到当前一级分类下的二级分类名称，以便快速理解品类差异。

验收要点：
- 右侧顶部展示当前一级分类名称。
- 二级分类按三列宫格展示，卡片仅展示名称，名称单行省略。
- 二级分类按后台排序升序排列。
- 不展示图片、商品数量、简介、价格、收藏或筛选控件。

## US-003 进入分类商品列表

作为访客，我希望点击二级分类后进入对应商品列表，以便继续浏览该分类下的瓷砖 SKU。

验收要点：
- 二级分类卡片整块可点击，并有原生按压反馈。
- 点击跳转 `pages/product-list/index?categoryId={secondaryCategoryId}&categoryName={encodedName}`。
- 300ms 内重复点击只触发一次跳转。
- 从商品列表或详情返回时，恢复当前一级分类和左右栏滚动位置。

## US-004 异常和缓存下继续浏览

作为访客，我希望网络异常或分类为空时页面仍然稳定，以便不被空白页打断。

验收要点：
- 有缓存时优先展示缓存并静默刷新。
- 无缓存加载失败时展示重试状态。
- 二级分类为空时保留当前一级分类标题并显示空状态。
- 二级分类不渲染图片资源，名称和点击能力保持可用。
