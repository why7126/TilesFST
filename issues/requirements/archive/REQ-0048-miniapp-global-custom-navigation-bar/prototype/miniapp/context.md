# REQ-0048 小程序全局自定义导航栏原型说明

## 1. 原型目标

本原型用于表达首页与非首页的统一导航结构，不作为最终视觉稿。后续实现应以 `REQ-0042`、`REQ-0043` 的深色品牌视觉为基准，并在微信开发者工具或真机中校验状态栏、胶囊避让和 fixed header 内容避让。

## 2. 页面形态

| 形态 | 说明 |
|---|---|
| 首页 | 保留当前品牌 `brand-header`；左侧无返回按钮；右侧避让微信原生胶囊。 |
| 非首页 | 左侧返回按钮；中间展示页面标题或品牌短标题；右侧避让微信原生胶囊。 |

## 3. 布局关键点

- 使用状态栏高度 + 胶囊高度 + 导航内容高度形成顶部 header。
- 导航栏 fixed 时，页面内容使用同一 header offset 下移。
- 右侧胶囊区域为不可占用区域，原型中以虚线占位表达。
- 返回按钮热区不小于 44x44 pt。
- 标题单行截断，不挤压右侧胶囊避让区。
