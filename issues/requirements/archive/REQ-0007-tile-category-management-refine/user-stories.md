## US-001 后台管理员更安全、一致地管理类目列表

**作为** 后台管理员或内部员工，  
**我希望** 类目启停有确认步骤、列表区更简洁、分页与用户管理页一致，  
**以便** 减少误操作，并在各管理列表页获得统一体验。

### 验收要点

- 点击「启用」「停用」时弹出确认框；取消不调用 API；确认后 Toast 并刷新树与列表。
- 检索区无「类目检索」标题行；筛选表单与查询/重置保留。
- 列表区无「类目列表」外层标题；表格工具栏（树上下文 + 共 N 条记录 + 调整排序）保留。
- 分页左侧「共 x 个类目」，右侧页码 +「每页显示 x 条」；无「当前显示 x-y / N 条」。
- 删除、编辑、新增弹窗及 BUG-0001 启停按钮可见性无回归。
