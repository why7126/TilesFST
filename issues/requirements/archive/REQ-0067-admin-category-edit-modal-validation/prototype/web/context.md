# 原型上下文

## 目标

该原型用于表达管理端类目新增 / 编辑弹窗在 `REQ-0067` 下的字段顺序、必填标识、编码隐藏和校验提示位置，不作为最终视觉稿。

## 字段

| 字段 | 新增 | 编辑 | 说明 |
|---|---|---|---|
| 上级类目 | 必填选择 | 不可修改 | 顶级类目用「无上级，创建一级类目」表达 |
| 类目名称 | 必填 | 必填 | 最多 10 字符，仅中文/英文/数字 |
| 类目编码 | 不展示 | 不展示可编辑项 | 后端自动生成 `CAT-` 前缀编码 |
| 排序权重 | 必填 | 必填 | 正整数 |
| 类目描述 | 可选 | 可选 | 最多 200 字，沿用现有规则 |
| 状态 | 沿用现有创建态 | 沿用现有规则 | 不改变启停专用流程 |

## 状态

- 默认态：显示必填标识和帮助文案。
- 字段错误态：错误提示在字段下方。
- 保存中：保存按钮禁用。
- 成功态：关闭弹窗并刷新类目树、列表、统计。
- 失败态：弹窗保持打开，表单值不丢失。

## 横切关注

- 专属弹窗类不得与通用 `modal-card` 双挂载。
- 1440px 视口 computed width 默认 560px。
- 矮视口下 body 可滚动，footer 不遮挡字段。
- CSS 使用 semantic token，不新增裸 Hex。
