# 原型说明

## 目标

本原型用于表达微信小程序品牌入口页与品牌主页/详情页的信息架构。入口页包含与首页一致的品牌轮播和一行 2 个品牌卡片；详情页上半部分展示品牌图片、品牌名称和品牌介绍，下半部分通过“商品 / 证书”Tab 展示当前品牌关联内容。

## 页面

| 页面 | 说明 |
|---|---|
| 品牌入口页 | 顶部品牌轮播，下方双列品牌卡片 |
| 品牌主页/详情页 | 品牌信息区 + 商品 / 证书 Tab |

## 视觉原则

- 使用小程序暗色旗舰风，避免浅色卡片破坏整体材质感。
- 品牌轮播对齐首页轮播，不独立创造新的轮播样式。
- 品牌卡片一行 2 个，品牌图片区域比例稳定。
- 详情页品牌信息区必须在首屏形成品牌识别点。
- 商品列表对齐 `REQ-0056` 双列商品卡片策略。
- 证书列表使用清晰缩略图和证书类型标签。

## 交互关注

- 品牌轮播和品牌卡片点击进入品牌主页。
- 品牌主页默认展示商品 Tab。
- 切换证书 Tab 时保留上方品牌信息。
- 详情页分享直达时返回按钮要有兜底。
- 加载、空态、错误态不被自定义导航或 TabBar 遮挡。
