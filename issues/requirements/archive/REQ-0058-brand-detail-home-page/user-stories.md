# 用户故事

## US-001 从品牌入口浏览品牌

作为装修客户，我希望在小程序中看到品牌入口页，通过轮播和双列品牌卡片快速了解可选品牌，以便按品牌开始浏览商品。

验收要点：

- 品牌入口页顶部展示与首页轮播一致的品牌轮播。
- 品牌列表一行 2 个品牌卡片。
- 品牌卡片展示品牌图片或 Logo、品牌名称，并可点击进入品牌主页/详情页。
- 品牌为空、加载失败或图片缺失时有稳定状态反馈。

## US-002 查看单品牌信息

作为设计师，我希望进入品牌主页后先看到品牌图片、品牌名称和品牌介绍，以便判断品牌调性是否适合客户方案。

验收要点：

- 品牌主页上半部分突出品牌图片和品牌信息。
- 品牌图片缺失时有合理兜底，不展示破图。
- 品牌不存在、禁用或不可公开时展示可恢复错误态。
- 长品牌名和长介绍不遮挡后续 Tab 内容。

## US-003 按品牌浏览商品

作为门店导购，我希望在品牌主页的商品 Tab 中看到当前品牌下的商品列表，以便向客户连续讲解同品牌商品。

验收要点：

- 商品 Tab 默认展示当前品牌下商品。
- 商品列表复用一行 2 个商品卡片展示策略。
- 商品卡片点击进入 SKU 详情页。
- 商品 Tab 支持分页、刷新、加载更多、空态和错误重试。

## US-004 查看品牌证书背书

作为普通访客，我希望在品牌主页的证书 Tab 中查看该品牌的证书，以便确认品牌资质和可信度。

验收要点：

- 证书 Tab 展示当前品牌关联且可公开的证书。
- 证书项展示证书图片、名称、类型等核心信息。
- 无证书时展示品牌上下文空态。
- 不可公开或不应展示的证书不出现在前台列表中。

## US-005 通过不同入口稳定返回和浏览

作为分享或跳转进入品牌页的用户，我希望页面导航、返回和内容 offset 稳定，以便不会被顶部导航、胶囊或底部 TabBar 遮挡。

验收要点：

- 品牌入口页和品牌主页按页面形态接入小程序自定义导航。
- 分享直达或异常参数直达时不白屏。
- 320/375/430 pt 视口下标题、返回、胶囊 reserve 和内容不重叠。
- 关键页面 `.ts` 与 `.js` 运行入口保持同步或有静态测试兜底。
