# 业务流程

## 1. 主流程

```text
用户进入小程序
  |
  +-- 首页 / 分类 / 运营入口
        |
        v
  品牌入口页
        |
        +-- 加载品牌轮播
        |     |
        |     +-- 有轮播：展示 swiper，点击进入品牌主页
        |     +-- 无轮播：展示降级视觉或隐藏轮播区
        |
        +-- 加载品牌列表
              |
              +-- 有品牌：一行 2 个品牌卡片
              +-- 无品牌：品牌空态
              +-- 失败：错误态 + 重试
                    |
                    v
              点击品牌卡片
                    |
                    v
              品牌主页/详情页
                    |
                    +-- 上半部分：品牌图片 + 品牌名称 + 品牌介绍
                    |
                    +-- 下半部分：Tab
                          |
                          +-- 商品 Tab：当前品牌商品列表 -> SKU 详情页
                          |
                          +-- 证书 Tab：当前品牌证书列表 -> 证书预览或详情
```

## 2. 异常流程

```text
品牌主页加载
  |
  +-- brandId 缺失 / 非法
  |     +-- 展示参数错误态，提供返回或回首页
  |
  +-- 品牌不存在 / 禁用 / 不可公开
  |     +-- 展示不可访问错误态，提供返回或回首页
  |
  +-- 网络失败
  |     +-- 保留页面骨架，展示重试入口
  |
  +-- 商品 Tab 无数据
  |     +-- 展示“该品牌暂无商品”类空态
  |
  +-- 证书 Tab 无数据
        +-- 展示“该品牌暂无证书”类空态
```

## 3. 与父需求差异

| 项 | REQ-0005-brand-management | REQ-0058-brand-detail-home-page |
|---|---|---|
| 终端 | Web 管理端 | 微信小程序 |
| 用户 | 企业内部员工 | 装修客户、设计师、门店导购、普通访客 |
| 核心职责 | 维护品牌主数据、Logo、介绍、状态和排序 | 前台展示品牌、商品集合和证书背书 |
| 数据动作 | 新增、编辑、启用、停用、删除、上传 Logo | 只读展示、跳转、分页、刷新、预览 |
| 权限边界 | 管理端鉴权 | 仅展示可公开品牌、商品和证书 |

## 5. 数据与接口关注点

- 品牌入口页需要可公开品牌列表和品牌轮播数据。
- 品牌主页需要按 `brandId` 获取品牌公开信息。
- 商品 Tab 需要按 `brandId` 获取公开 SKU 列表。
- 证书 Tab 需要按 `brandId` 获取公开证书列表。
- 若现有 API 已能支撑，应优先复用；若需要新增或调整 API，必须在后续 `/req-opsx` 中同步 OpenAPI、Orval、docs 和 tests。
