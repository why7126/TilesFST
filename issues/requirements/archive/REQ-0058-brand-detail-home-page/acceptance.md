# 验收标准

## 功能 AC

- [ ] AC-001 小程序存在品牌入口页，可从首页、分类或运营入口之一进入。
- [ ] AC-002 品牌入口页顶部展示品牌轮播，视觉、指示器和滑动交互与首页轮播一致。
- [ ] AC-003 品牌轮播项点击后可进入对应品牌主页/详情页。
- [ ] AC-004 无品牌轮播数据时页面展示降级视觉或隐藏轮播区，不得白屏。
- [ ] AC-005 品牌入口页展示品牌列表，品牌卡片一行 2 个。
- [ ] AC-006 品牌卡片展示品牌图片或 Logo、品牌名称。
- [ ] AC-007 品牌图片或 Logo 缺失时展示统一占位，不展示破图。
- [ ] AC-008 品牌名称过长时限制行数或截断，不撑破卡片。
- [ ] AC-009 品牌卡片整卡可点击，进入对应品牌主页/详情页。
- [ ] AC-010 品牌入口页支持加载中、空状态、错误状态和重试。

## 品牌主页/详情页 AC

- [ ] AC-011 品牌主页/详情页支持通过 `brandId` 路由参数访问。
- [ ] AC-012 页面上半部分展示品牌图片或 Logo、品牌名称和品牌介绍。
- [ ] AC-013 品牌图片优先使用品牌主图；无主图时可回退到品牌 Logo。
- [ ] AC-014 品牌介绍为空时隐藏对应区域或展示简短兜底，不展示异常空字段。
- [ ] AC-015 品牌不存在、禁用、下架或不可公开时展示可恢复错误态，不得白屏。
- [ ] AC-016 品牌主页下半部分展示 Tab 内容区，首期包含“商品”和“证书”。
- [ ] AC-017 默认打开“商品”Tab。
- [ ] AC-018 切换 Tab 时上半部分品牌信息保持稳定，不丢失或异常刷新。
- [ ] AC-019 每个 Tab 的加载中、空状态、错误状态和重试互相独立。

## 商品 Tab AC

- [ ] AC-020 商品 Tab 仅展示当前品牌下的公开商品。
- [ ] AC-021 商品列表采用一行 2 个商品卡片展示，复用或对齐 `REQ-0056` 的卡片策略。
- [ ] AC-022 商品卡片展示主图、商品名称、品牌或规格、参考价格和状态徽标。
- [ ] AC-023 商品卡片点击进入 SKU 详情页，传参正确。
- [ ] AC-024 商品 Tab 支持首屏加载、下拉刷新、上拉加载更多、无更多和加载更多失败重试。
- [ ] AC-025 当前品牌无商品时展示品牌上下文空态。

## 证书 Tab AC

- [ ] AC-026 证书 Tab 仅展示当前品牌关联且可公开的证书。
- [ ] AC-027 证书项展示证书图片、证书名称、证书类型和必要的有效状态。
- [ ] AC-028 证书图片加载失败时展示统一占位，不展示破图。
- [ ] AC-029 证书项点击支持预览证书图片或进入证书详情；最终交互需在 OpenSpec Change 中确认。
- [ ] AC-030 当前品牌无证书时展示品牌上下文空态。
- [ ] AC-031 不可公开、已下架或不应展示的证书不出现在前台列表中。

## UI / 设备 / 埋点 AC

- [ ] AC-032 页面视觉延续小程序“工业石材 · 暗色旗舰风”，品牌金仅用于强调、激活态和关键 CTA。
- [ ] AC-033 品牌入口页首屏同时露出品牌轮播和至少一部分品牌卡片内容。
- [ ] AC-034 品牌主页首屏以品牌图片和品牌名称形成明显识别点。
- [ ] AC-035 长品牌名、长商品名、长证书名均截断或限制行数，不遮挡相邻内容。
- [ ] AC-036 品牌入口页和品牌主页接入自定义导航时，状态栏、胶囊 reserve、返回按钮和内容 offset 按 `docs/knowledge-base/best-practices/miniapp-custom-navigation.md` 验收。
- [ ] AC-037 DevTools 320/375/430 pt 视口下，品牌轮播、品牌卡片、Tab、商品卡片、证书卡片与底部 TabBar 不重叠。
- [ ] AC-038 真机验收不可用时，必须在验收材料中标记 blocked 或 follow_up，不得写作真机通过。
- [ ] AC-039 新增小程序页面运行入口 `.ts` / `.js` 保持同步或有静态测试兜底，避免 DevTools 实际加载入口与源码漂移。
- [ ] AC-040 品牌卡片点击、轮播点击、品牌主页浏览、Tab 切换、商品点击和证书点击埋点不阻断用户操作。

## 横切 AC（knowledge-base）

无管理端横切标签命中。本需求为微信小程序前台 UI，不涉及 `admin-list`、`admin-form`、`admin-modal` 或 `media-upload`，因此不新增 `AC-XCUT-*`。小程序导航与设备验收风险已作为 AC-036 至 AC-039 写入。
