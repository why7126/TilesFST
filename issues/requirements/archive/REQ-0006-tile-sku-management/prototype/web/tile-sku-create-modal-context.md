# 瓷砖SKU管理页面 - 新增SKU弹窗原型上下文 v4

## 1. 原型目标

本文件用于指导 Cursor / AI 前端开发还原「新增SKU」弹窗状态。开发时应优先参考 `prototype/web/tile-sku-create-modal.html`；`prototype/images/tile-sku-create-modal.png` 为**可选** Golden Reference（有则辅助验收，无则以 HTML 为准）。

## 2. 弹窗设计原则

- 弹窗覆盖在 SKU 列表页之上，背景使用半透明遮罩。
- 弹窗宽度建议 880px，高度不超过视口，内部滚动。
- 弹窗使用暗色卡片、0.5px 细边框、2-3px 圆角。
- 新增 SKU 默认保存为草稿，弹窗内不出现状态字段。

## 3. 表单字段顺序

```text
SKU名称
SKU编码
所属品牌
所属类目
规格尺寸
表面工艺
主色系
参考价格（元）
SKU图片（多张，设置主图）
SKU视频（多个）
备注说明
```

## 4. 图片上传要求

- SKU 图片支持多张。
- 图片区域展示缩略图网格。
- 必须支持指定一张为主图。
- 主图缩略图左上角展示「主图」标签。
- 非主图展示「设为主图」操作。
- 支持继续添加图片。

## 5. 视频上传要求

- SKU 视频支持多个。
- 视频以文件卡片展示，展示文件名、大小、上传状态。
- 支持继续添加视频。
- 视频不是必填项。

## 6. 价格字段要求

- 字段 Label 必须是「参考价格（元）」。
- 输入框 Placeholder：请输入参考价格。
- 支持小数，保存时按金额数值处理。

## 7. 操作按钮

弹窗底部右侧按钮：

- 取消：幽灵按钮。
- 保存草稿：描边按钮。
- 创建SKU：品牌金实底主按钮。

## 8. 一致性检查清单

- [ ] 弹窗内不出现状态字段。
- [ ] 价格字段名称为「参考价格（元）」。
- [ ] 图片区域支持多张并展示主图标记。
- [ ] 视频区域支持多个视频卡片。
- [ ] 弹窗底部操作为取消、保存草稿、创建SKU。
- [ ] 视觉继承管理后台首页暗色旗舰风。
