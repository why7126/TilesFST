## US-001 运营人员进入管理端工作台

**作为** 企业内部运营人员，  
**我希望** 登录后看到简洁、专业的管理后台首页，  
**以便** 快速了解核心数据概况并进入高频新建操作。

### 验收要点

- 登录成功后进入 `/admin/dashboard`，页面布局与 `prototype/web/admin-home.png` 一致。
- 左侧 Sidebar 固定 264px 宽、100vh 高，不随右侧内容滚动。
- 右侧内容区独立滚动，包含数据概览、快捷操作、最近更新三个模块。
- 数据概览展示 SKU 总数、品牌数量、Banner 数量、用户数量四个指标卡（本期可为 mock 数据）。
- 快捷操作仅包含：新增 SKU、新增品牌、新增类目、新增 Banner。
- 最近更新以表格展示更新时间、类型、名称、操作人（本期可为 mock 数据）。

### 关联功能

- FR-001：管理端整体布局
- FR-002：数据概览模块
- FR-003：快捷操作模块
- FR-004：最近更新模块

## US-002 管理员通过 Sidebar 导航与用户菜单管理系统

**作为** 系统管理员，  
**我希望** 通过固定 Sidebar 浏览各管理模块，并通过底部用户菜单访问个人设置与退出登录，  
**以便** 在统一导航框架下高效切换功能并保持操作安全。

### 验收要点

- Sidebar 品牌名统一为 **TILESFST**，不出现 STONEX。
- OPERATIONS 分组包含：首页、瓷砖 SKU、瓷砖品牌、瓷砖类目、Banner 管理。
- SYSTEM 分组包含：用户管理、系统设置。
- 用户菜单固定在 Sidebar 底部，展示头像缩写、用户名、邮箱与展开箭头。
- 用户菜单下方不得直接展示「退出登录」按钮。
- 点击用户菜单弹出下拉框，包含：个人资料、密码修改、分隔线、退出登录（风险色弱强调）。
- 点击退出登录可调用现有 auth logout 并跳转 `/admin/login`。
- 个人资料、密码修改为占位入口，本期可不实现完整流程。

### 关联功能

- FR-005：Sidebar 导航分组
- FR-006：用户菜单与下拉框
- FR-007：退出登录（复用 REQ-0001 认证能力）

## 跨角色通用期望

| 期望 | 说明 |
|---|---|
| 视觉一致性 | 延续登录页「工业石材 · 暗色旗舰风」，HTML 原型为最高优先级视觉参考 |
| 导航占位 | 除首页外，Sidebar 链接本期可为占位路由或 `#`，不阻塞首页验收 |
| 快捷操作占位 | 四个「新增」入口本期可跳转占位页或展示「功能建设中」 |
| 响应式 | 桌面端为主验收视口（1440×1024）；小于 1024px 按原型 context 降级 |
