# REQ-0004 管理后台首页 V5 需求文档

## 1. 需求背景

TILESFST 管理后台首页用于企业内部运营人员进入系统后的工作台入口。页面需要延续登录页与全局 `ui-design.md` 的「工业石材 · 暗色旗舰风」视觉语言，保持暗色背景、克制金色、极细分割线、近直角圆角与专业后台质感。

本版本在 V4 基础上继续精简首页快捷操作，仅保留真实高频的新建类入口，删除「导入 SKU、导入图片、价格管理、操作日志」四个快捷操作项。

## 2. 设计目标

- 构建简洁、稳定、专业的管理后台首页。
- 左侧导航栏固定高度，不随右侧内容高度变化，交互参考 ChatGPT.com。
- 用户菜单固定在左侧导航底部。
- 退出登录不直接展示在底部，而是收纳在用户菜单下拉框中。
- 右侧内容仅保留核心工作台模块：数据概览、快捷操作、最近更新。
- 快捷操作仅保留 4 个：新增 SKU、新增品牌、新增类目、新增 Banner。

## 3. 页面范围

### 3.1 包含范围

- 管理后台首页整体布局。
- 固定左侧导航栏。
- 左侧用户菜单与下拉框。
- 右侧数据概览。
- 右侧快捷操作。
- 右侧最近更新表格。

### 3.2 不包含范围

- 登录认证逻辑。
- 真实数据接口。
- SKU、品牌、类目、Banner 的详情页实现。
- 用户资料、密码修改、退出登录的业务流程实现。

## 4. 页面布局

页面采用左右布局：

- 左侧：固定宽度 Sidebar，宽度 264px，高度 100vh。
- 右侧：内容区独立滚动，高度 100vh。
- Sidebar 不随着右侧内容高度而拉伸或滚动。

```text
┌──────────────┬──────────────────────────────┐
│ TILESFST     │ 数据概览                     │
│              │ 快捷操作                     │
│ OPERATIONS   │ 最近更新                     │
│ SYSTEM       │                              │
│              │                              │
│ 用户菜单     │                              │
└──────────────┴──────────────────────────────┘
```

## 5. 左侧导航栏

### 5.1 品牌

品牌名统一为：

```text
TILESFST
```

不得再出现 STONEX。

### 5.2 导航分组

#### OPERATIONS

- 首页
- 瓷砖 SKU
- 瓷砖品牌
- 瓷砖类目
- Banner 管理

#### SYSTEM

- 用户管理
- 系统设置

### 5.3 用户菜单

用户菜单固定在 Sidebar 底部，展示：

- 头像缩写：AU
- 用户名：Admin User
- 邮箱：admin@tilesfst.com
- 向上箭头 / 展开状态标识

底部用户菜单下方不得直接展示「退出登录」按钮。

### 5.4 用户菜单下拉框

点击用户菜单后弹出下拉框，位于用户菜单上方。下拉项包括：

1. 个人资料
2. 密码修改
3. 分隔线
4. 退出登录

退出登录需要与上方两个菜单项通过 0.5px 分隔线区分，并使用风险色弱强调。

## 6. 右侧首页内容

### 6.1 删除内容

本页面不得出现以下模块：

- 顶部欢迎区域
- 今日待办
- 数据质量概览
- 风险提醒
- 热门材质
- 门店同步
- 材质库存分布

### 6.2 数据概览

展示 4 个核心指标卡：

- SKU 总数：12,860
- 品牌数量：128
- Banner 数量：36
- 用户数量：42

### 6.3 快捷操作

快捷操作采用 4 宫格，仅包含：

- 新增 SKU
- 新增品牌
- 新增类目
- 新增 Banner

必须删除以下快捷操作项：

- 导入 SKU
- 导入图片
- 价格管理
- 操作日志

### 6.4 最近更新

最近更新采用表格展示，字段包括：

- 更新时间
- 类型
- 名称
- 操作人

## 7. UI/UE 约束

- 页面背景使用 `#18160F`。
- 卡片背景使用 `#211E16` 或半透明等效层。
- 品牌金使用 `#C8A055`，仅用于 Logo、激活态、关键数字和轻量强调。
- 分割线使用 `rgba(255,255,255,0.07)` 或 `rgba(255,255,255,0.1)`。
- 圆角控制在 2px–3px。
- Sidebar 高度必须为 `100vh`。
- 右侧内容区域独立滚动。
- HTML Golden Reference 是前端开发最高优先级视觉参考。
