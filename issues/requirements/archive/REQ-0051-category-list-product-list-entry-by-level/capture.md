# 分类列表页支持一二级分类商品列表入口

分类列表页允许用户进入一级分类对应的商品列表页，也允许用户进入二级分类对应的商品列表页，确保不同层级分类都能作为商品浏览入口。

# 原始描述

分类列表页，允许用户进入【一级分类】的商品列表页，也允许用户进入二级分类的商品列表页

# 待澄清

- [ ] 该能力优先适用于微信小程序、店主 Web 展示端，还是多端统一支持。
- [ ] 一级分类商品列表页是否展示该一级分类下所有二级分类商品，还是仅展示直接挂载在一级分类下的商品。
- [ ] 二级分类商品列表页是否复用现有商品列表通用组件、筛选、搜索、分页和空状态。
- [ ] 分类列表页进入商品列表页时需要传递哪些参数，例如 `category_id`、层级、分类名称或面包屑。
- [ ] 页面标题、导航返回、分类路径展示和无商品状态是否需要按一级/二级分类区分。
