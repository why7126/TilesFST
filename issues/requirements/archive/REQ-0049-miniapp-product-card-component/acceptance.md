# 验收标准

## 功能 AC

- [ ] AC-001 商品卡片组件可被分类商品列表、搜索结果 SKU 列表和首页推荐商品区引用，且不在组件内部维护列表查询、筛选、排序或分页状态。
- [ ] AC-002 商品卡片展示主图、SKU 名称、品牌、规格和参考价格；SKU 名称最多两行并在超出时省略。
- [ ] AC-003 参考价格使用“参考价格”语义；价格缺失时展示统一兜底文案，不展示空字符串、`null`、`undefined` 或错误金额。
- [ ] AC-004 商品主图使用固定比例容器；主图缺失或加载失败时展示统一占位，不出现破图或卡片高度跳动。
- [ ] AC-005 整张商品卡片可点击；点击后跳转 SKU 详情页并携带 `skuId` 与来源上下文。
- [ ] AC-006 卡片点击具备防重复触发能力，连续点击不会重复打开多个详情页。
- [ ] AC-007 商品不可查看或缺少 `skuId` 时阻止跳转，展示不可用提示或不可用状态。
- [ ] AC-008 单张卡片字段缺失、图片失败或不可查看时，不影响同一列表中其他商品卡片展示。
- [ ] AC-009 卡片在 320px 小屏宽度下保持文本可读，点击区域有效尺寸不小于 44px。
- [ ] AC-010 v1 卡片内不提供收藏、分享、加入询价、购物车或联系客服快捷按钮。
- [ ] AC-011 卡片支持曝光、点击、图片失败和不可用点击埋点；参数包含 `skuId`、`sourcePage`、`sourceModule`、`listContext`、`index` 和 `requestId`。
- [ ] AC-012 视觉风格与菲尚特小程序深色企业轻奢风一致，价格和激活态使用品牌金语义，卡片圆角、分割线和触控反馈统一。

## 场景 AC

- [ ] AC-013 分类商品列表从 `categoryId` 上下文渲染商品卡片时，卡片点击进入对应 SKU 详情页。
- [ ] AC-014 搜索结果页从 `keyword` 上下文渲染商品卡片时，卡片点击保留搜索来源。
- [ ] AC-015 首页推荐模块复用商品卡片时，可传入推荐模块来源与展示位置。
- [ ] AC-016 后续品牌商品列表或收藏列表复用商品卡片时，不需要复制核心展示结构。
