# 业务流程

## 1. 总流程

```text
管理端证书数据
  |
  v
页面 / 弹窗 / 后续后台入口
  |
  +-- 列表证书列 --------> CertificateThumb + CertificateSummary
  |
  +-- 有效期与状态列 ----> CertificateValidityText + CertificateValidityBadge
  |
  +-- 展示状态列 --------> CertificateVisibilityBadge
  |
  +-- 预览操作 ----------> CertificatePreviewAction
  |
  +-- 上传文件区域 ------> CertificateFileCard
                            |
                            v
                    页面容器负责接口、权限、保存、确认和 toast
```

## 2. 列表展示流程

| 步骤 | 说明 | 输出 |
|---|---|---|
| 1 | 品牌证书管理页请求证书列表 | 获得 `BrandCertificateItem` 数据 |
| 2 | “证书”列调用通用缩略图与信息单元 | 展示缩略图/PDF/占位、证书名、证书编号或文件名 |
| 3 | 有效期列调用通用有效期格式化 | 展示长期有效、日期区间或未设置 |
| 4 | 有效状态列调用通用 Badge | 展示有效、即将到期、已过期等状态 |
| 5 | 前台展示列调用展示状态 Badge | 展示展示中或已隐藏 |
| 6 | 操作列保留页面行为 | 编辑、显示/隐藏、删除仍由页面控制 |

## 3. 预览流程

```text
用户点击缩略图或预览入口
  |
  v
组件检查文件 URL
  |
  +-- URL 存在 --> 页面按图片/PDF 预览策略打开新窗口
  |
  +-- URL 缺失 --> 返回失败原因
                     |
                     v
               页面展示预览失败提示
```

## 4. 文件卡片流程

| 状态 | 触发 | 展示 |
|---|---|---|
| `idle` | 新建证书尚未选择文件 | 未上传提示、选择/拖拽入口 |
| `uploading` | 页面容器正在上传文件 | 文件名、进度、取消或禁用保存提示 |
| `done` | 上传成功或编辑已有文件 | 缩略图/PDF/文件占位、文件名、大小、重新上传、删除 |
| `failed` | 上传失败或文件校验失败 | 失败原因、重新上传入口 |

文件卡片只负责展示和回调，上传 API、进度计算、错误映射和保存阻塞仍由页面容器负责。

## 5. 与父需求差异

| 条目 | REQ-0038 品牌证书管理页 | REQ-0055 品牌证书通用组件 |
|---|---|---|
| 目标 | 完成完整品牌证书管理页面和 CRUD 能力 | 抽取证书展示与文件卡片组件 |
| API / DB | 包含品牌证书接口、上传、数据关系和审计要求 | 不新增 API / DB，只复用已有字段 |
| 页面范围 | `/admin/brand-certificates` 一级页面 | 优先应用于现有页面内的列和弹窗局部 |
| 权限与确认 | 页面负责权限、确认、显示/隐藏、删除 | 组件不内置权限或状态变更接口 |
| 上传链路 | 定义上传类型、大小、后端链路和保存规则 | 只沉淀文件卡片状态展示和回调契约 |

## 6. 非目标流程

- 不进入店主 Web 品牌详情页证书展示流程。
- 不进入微信小程序证书展示流程。
- 不处理证书审批、OCR、真伪验证、电子签章或批量操作。
- 不改变品牌证书上传 API、MinIO 前缀、文件大小限制或数据库字段。
