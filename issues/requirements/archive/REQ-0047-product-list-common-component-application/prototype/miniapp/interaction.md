# 商品列表页交互说明 v1

## 进入

- 分类页二级分类 → 分类商品列表。
- 搜索结果 SKU Tab → 搜索商品列表。
- 首页推荐模块 → 推荐商品列表。
- 品牌相关页面 → 品牌商品列表。

## 搜索

搜索默认作用于当前列表上下文；搜索结果 SKU 列表必须保留关键词，筛选和排序不得丢失关键词。

## 筛选

```text
点击筛选入口
  |
  v
打开底部抽屉
  |
  +-- 重置 --> 清空筛选草稿
  |
  +-- 确认 --> 刷新列表第一页
```

## 商品浏览

- 商品卡片展示主图、商品名称、SKU 编码、品牌、规格和参考价格。
- 商品图片区域比例稳定，加载前后不造成明显跳动。
- 滚动到底部自动加载下一页，加载中防止重复请求。

## 商品点击

点击任意商品卡片区域进入 SKU 详情页；v1 不在卡片内提供收藏、询价、购物车或联系商家按钮。

## 空状态

- 分类无商品：展示“该分类暂未上架商品”。
- 搜索无结果：展示当前关键词、调整建议和清空筛选入口。
- 筛选无匹配：展示清空筛选入口。

## 运营插槽

v1 不包含后台可配置运营插槽；首页推荐或品牌入口只作为商品列表上下文来源。
