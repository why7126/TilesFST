# 原型上下文

## 1. 原型用途

本目录中的 `banner-management-list-logo.html` 与 PNG 用于验证管理端 Sidebar 顶部品牌区，不代表本需求要修改 Banner 管理业务页面。

## 2. 还原优先级

1. Sidebar 品牌区布局与 `banner-management-list-logo.html` / PNG 保持一致。
2. 品牌文案以 `requirement.md` 和 `acceptance.md` 为准：
   - 主标题：`菲尚特FST`
   - 副标题：`家居建材资料库`
3. 若旧原型上下文出现 `家居建材管理后台`，视为历史文案，实施时必须替换为 `家居建材资料库`。

## 3. 关键约束

- Logo、品牌文字组、展开/收起按钮同一行。
- Logo 不加独立背景、边框、底纹或阴影。
- 版本号靠近 `菲尚特FST` 主标题右上侧。
- 收起按钮为正方形热区，与 Sidebar 右边界保留安全间距。
- favicon 使用菲尚特 Logo 或由该 Logo 派生的 Web 图标资源。
