# REQ-0050 小程序 brand-header 页面标题规则业务流程

## 1. 页面进入与标题决策

```text
用户进入小程序页面
  |
  v
页面是否首页 index？
  |
  +-- 是：首页 brand-header
  |     |
  |     +-- 第一行：菲尚特瓷砖馆
  |     +-- 第二行：质感空间，由砖而生
  |     +-- 不显示返回按钮
  |     +-- 右侧避让微信原生胶囊
  |     +-- 主体内容从导航栏下方开始
  |
  +-- 否：非首页 brand-header
        |
        +-- 左侧显示返回按钮
        +-- 中间展示一行页面标题
        +-- 不展示 subtitle / 第二行副文案
        +-- 右侧避让微信原生胶囊
        +-- 主体内容从导航栏下方开始
```

## 2. 非首页返回流程

```text
用户点击非首页返回按钮
  |
  v
是否存在上一页页面栈？
  |
  +-- 是：navigateBack(delta=1)
  |
  +-- 否：兜底进入首页
        |
        +-- 优先 switchTab / 首页路由
        +-- 失败时 reLaunch 首页
```

## 3. 页面标题映射

| 页面 | 导航形态 | 标题规则 | 第二行 |
|---|---|---|---|
| index | 首页 | `菲尚特瓷砖馆` + `质感空间，由砖而生` | 有 |
| search | 非首页 | `搜索` | 无 |
| category | 非首页 | `全部分类` | 无 |
| product-list | 非首页 | `全部商品` 或列表上下文标题 | 无 |
| tile-detail | 非首页 | `商品详情` | 无 |
| favorites | 非首页 | `收藏` | 无 |
| certificates | 非首页 | `证书` | 无 |
| store-info | 非首页 | `门店信息` | 无 |

## 4. 与父 REQ 差异

| 对比项 | REQ-0048 全局自定义导航栏 | REQ-0050 页面标题规则 |
|---|---|---|
| 关注点 | 导航模块全局化、返回、状态栏和胶囊避让 | 首页 / 非首页文案规则收束 |
| 首页 | 保留品牌 `brand-header` | 固定两行：`菲尚特瓷砖馆`、`质感空间，由砖而生` |
| 非首页 | 带返回按钮并展示标题 / 品牌信息 | 只能显示一行页面标题，禁止第二行 subtitle |
| 详情页 | 需要接入非首页导航 | 标题本期固定为 `商品详情`，动态 SKU 信息留在内容区 |
| OpenSpec 粒度 | 全局导航能力 | 对既有导航能力的规则 refinement |

## 5. 异常与兜底

| 场景 | 处理 |
|---|---|
| 从分享卡片直达详情页 | 显示 `商品详情` 单行标题；返回无栈时回首页；分享能力保留。 |
| 页面加载失败 | 导航栏仍展示页面标题与返回按钮；错误内容显示在导航栏下方。 |
| 标题过长 | 单行截断，不换行，不挤压右侧胶囊避让区。 |
| `find` / `profile` 后续接入自定义导航 | 默认按非首页单行标题规则处理，除非后续需求豁免。 |
| 微信胶囊信息获取失败 | 使用安全 fallback 尺寸，仍不得绘制伪分享 / 关闭按钮。 |

## 6. 风险控制

| 风险 | 控制方式 |
|---|---|
| 页面继续传入 subtitle 导致两行展示 | acceptance 明确非首页禁止第二行，后续实现可由组件忽略或移除 subtitle。 |
| 首页文案被接口 store.description 覆盖 | 首页品牌副文案固定为 `质感空间，由砖而生`，不由接口描述动态替换。 |
| 商品详情标题使用 SKU 导致过长 | 本期固定 `商品详情`，动态信息放入内容区。 |
| 胶囊遮挡标题 | 右侧预留微信原生胶囊安全区，标题单行截断。 |
| 内容被 fixed header 遮挡 | 统一 spacer / 导航高度机制覆盖正常、加载、空态、错误态。 |
