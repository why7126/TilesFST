Web / 管理端主题舒适度优化与主题切换

本原型展示四类主题模式、瓷砖 SKU 管理页验收矩阵、SKU 弹窗/上传状态，以及店主 Web 舒适主题边界。实现时必须使用 Design System semantic token，不直接复制本文件色值。

主题模式

系统默认 / 暗色旗舰 / 舒适暗色 / 浅色

系统默认auto

跟随系统或产品默认策略，必须解释最终映射。

暗色旗舰brand

保留工业石材品牌暗色，适合品牌展示和用户主动选择。

舒适暗色comfort

降低纯黑压迫和刺眼对比，优先服务管理端长时间操作。

浅色light

强光环境和偏好浅色用户使用，需完整 token 验收。
