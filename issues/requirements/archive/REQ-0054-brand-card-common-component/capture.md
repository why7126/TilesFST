# 生成品牌卡片通用组件

沉淀品牌卡片通用组件，用于统一品牌 Logo、品牌名称、品牌简介、状态/标签、跳转入口等品牌展示信息的布局、样式和交互，减少各页面重复实现并保持品牌信息展示一致。

# 原始描述

生成品牌卡片通用组件

# 待澄清

- [ ] 品牌卡片优先应用端是微信小程序、Web 店主展示端、管理端，还是需要跨端分别实现同名组件规范。
- [ ] 卡片必须展示的字段范围：品牌 Logo、品牌名称、英文名/别名、简介、产地、标签、证书数量、SKU 数、启用状态、推荐状态等是否都纳入首版。
- [ ] 卡片交互是否只支持点击跳转品牌详情，还是需要支持收藏、筛选、编辑、上下架等操作入口。
- [ ] 空状态与异常图片策略是否复用现有品牌 Logo fallback、占位图和图片加载失败样式。
- [ ] 是否需要同时沉淀组件验收用例，包括不同 Logo 比例、长品牌名、多标签、无简介、禁用品牌、移动端窄屏等场景。
