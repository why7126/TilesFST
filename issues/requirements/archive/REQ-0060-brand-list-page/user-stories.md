# 用户故事

## US-001 用户从品牌入口进入品牌列表页

作为装修业主或普通访客，我希望点击小程序中的“品牌”入口后进入品牌列表页，以便按品牌维度浏览菲尚特可展示的瓷砖品牌。

验收要点：

- 原“找砖”入口文案已调整为“品牌”。
- 点击“品牌”入口进入品牌列表页，而不是继续进入搜索页或找砖页。
- 页面标题、导航标题和入口文案保持“品牌”语义一致。
- 品牌列表页加载失败时展示可恢复错误态，不白屏。

## US-002 用户查看品牌轮播

作为访客，我希望在品牌列表页顶部看到品牌轮播，以便快速了解主推品牌、活动品牌或重点品牌内容。

验收要点：

- 品牌轮播与首页轮播保持一致的自动播放、循环播放和指示点体验。
- 轮播图加载失败时使用品牌化兜底或隐藏异常图片，不展示破图。
- 点击有效轮播项可进入配置目标；目标不可达时安全降级并提示。
- 轮播区域不遮挡小程序自定义导航、状态栏或微信原生胶囊。

## US-003 用户浏览双列品牌卡片

作为正在选材的用户，我希望品牌列表以一行 2 个卡片展示，以便在手机屏幕中快速扫视多个品牌。

验收要点：

- 品牌列表一行展示 2 个品牌卡片。
- 每张卡片至少展示品牌 Logo 和品牌名称。
- 卡片可展示品牌简介、标语、商品数量或“查看品牌主页”等辅助信息。
- 品牌 Logo 加载失败时显示可识别占位，不破坏卡片布局。
- 双列卡片在 320 到 430 pt 宽度下不出现横向滚动、文字重叠或卡片挤压。

## US-004 用户进入品牌详情或品牌商品列表

作为对某个品牌感兴趣的用户，我希望点击品牌卡片后继续查看品牌主页或该品牌相关商品，以便深入了解品牌和产品。

验收要点：

- 整张品牌卡片可点击，并有小程序原生按压反馈。
- 优先进入 `REQ-0058-brand-detail-home-page` 定义的品牌详情页/主页。
- 若品牌详情页未交付，可临时进入品牌商品列表并携带品牌筛选参数。
- 已下架、未公开或缺少有效跳转目标的品牌不得进入无效页面。

## US-005 产品团队追踪品牌页使用效果

作为产品运营人员，我希望记录品牌页曝光、轮播点击和品牌卡片点击，以便评估品牌频道的访问和转化效果。

验收要点：

- 品牌列表页曝光记录 `brand_list_page_view` 或等价事件。
- 品牌轮播点击记录轮播 ID、跳转类型和位置索引。
- 品牌卡片点击记录品牌 ID、位置索引和来源入口。
- 埋点不得记录手机号、地址、微信号等与品牌浏览无关的个人敏感信息。
