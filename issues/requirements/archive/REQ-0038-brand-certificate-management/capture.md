# 新增品牌证书管理页

为管理端新增品牌证书管理页，用于维护品牌相关证书资料，支持后续在品牌资料、店主展示端或内部审核流程中引用。

# 原始描述

新增品牌证书管理页

# 待澄清

- [ ] 证书归属范围：按品牌维护，还是支持品牌下多个证书主体或系列。
- [ ] 证书字段：证书名称、编号、发证机构、有效期、证书图片或文件、状态等是否需要纳入首版。
- [ ] 管理操作：是否需要新增、编辑、删除、启停、排序、预览、下载。
- [ ] 展示边界：证书是否对店主 Web 展示端或微信小程序可见，是否需要公开/内部可见状态。
- [ ] 上传约束：证书文件类型、大小、MinIO 前缀、鉴权和签名访问策略。
