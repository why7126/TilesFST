# 微信小程序多页面支持微信分享

# 原始描述

微信小程序 首页、商品详情页、商品列表页、品牌详情页支持分享给微信朋友以及微信朋友圈。

# 待澄清

- [ ] 分享给微信朋友与分享到朋友圈是否需要使用不同标题、图片、路径和参数。
- [ ] 首页、商品详情页、商品列表页、品牌详情页各自的分享标题、封面图、落地路径与参数规则。
- [ ] 商品详情页和品牌详情页是否需要携带商品/品牌 ID，并在被分享用户打开后准确回到对应详情页。
- [ ] 商品列表页分享是否需要保留当前分类、品牌、搜索词、排序等筛选上下文。
- [ ] 分享封面是否复用现有图片资源，或需要管理端配置默认分享图。
