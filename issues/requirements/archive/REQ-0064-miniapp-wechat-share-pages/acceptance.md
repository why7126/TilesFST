# 验收标准

## 1. 页面分享矩阵

- [ ] AC-001 首页支持分享给微信朋友，分享标题、路径和兜底标题符合 PRD。
- [ ] AC-002 首页支持分享到微信朋友圈，点击朋友圈入口后可进入首页。
- [ ] AC-003 商品详情页支持分享给微信朋友，分享路径保留当前 `skuId` 与 `source=share`。
- [ ] AC-004 商品详情页支持分享到微信朋友圈，分享直达后加载对应 SKU。
- [ ] AC-005 商品列表页支持分享给微信朋友，分享标题匹配当前列表语义。
- [ ] AC-006 商品列表页支持分享到微信朋友圈，分享直达后保留列表上下文。
- [ ] AC-007 品牌详情页支持分享给微信朋友，分享路径保留当前 `brandId`。
- [ ] AC-008 品牌详情页支持分享到微信朋友圈，分享直达后加载对应品牌。

## 2. 路径与参数

- [ ] AC-009 商品详情页分享路径必须保留有效 `skuId`，无效 `skuId` 不得白屏。
- [ ] AC-010 品牌详情页分享路径必须保留有效 `brandId`，无效 `brandId` 不得白屏。
- [ ] AC-011 商品列表页搜索分享必须保留并正确解码 `keyword`。
- [ ] AC-012 商品列表页分类分享必须保留 `categoryId`、`categoryLevel`、`categoryName` 中的适用参数。
- [ ] AC-013 商品列表页品牌商品分享必须保留 `brandId`。
- [ ] AC-014 商品列表页新品榜 / 热销榜分享必须保留 `section`。
- [ ] AC-015 中文分类名、品牌名、搜索词等 query 参数必须安全编码，分享后可恢复页面语义。

## 3. 分享标题与图片

- [ ] AC-016 首页分享标题优先使用门店 / 品牌名称，未加载时使用兜底标题。
- [ ] AC-017 商品详情页分享标题优先使用商品分享标题或 SKU 名称 + 品牌名称。
- [ ] AC-018 商品详情页分享图优先使用分享图或商品主图；缺失时使用安全兜底图。
- [ ] AC-019 商品列表页分享标题必须反映搜索、分类、品牌、榜单或全部商品语义。
- [ ] AC-020 品牌详情页分享标题优先使用品牌名称，未加载时使用兜底标题。
- [ ] AC-021 分享图片必须为小程序可访问的安全 URL 或本地资源，不得直连未授权对象存储地址。

## 4. 分享直达与异常态

- [ ] AC-022 从微信朋友分享直达四个目标页面时，页面加载态、正常态、空状态和错误态均不白屏。
- [ ] AC-023 从朋友圈分享直达四个目标页面时，页面加载态、正常态、空状态和错误态均不白屏。
- [ ] AC-024 商品详情、商品列表和品牌详情无页面栈直达时，返回按钮或等价入口必须可兜底回首页。
- [ ] AC-025 参数无效、资源下架或网络失败时，页面必须展示可理解文案和可操作恢复路径。
- [ ] AC-026 分享直达不得遮挡自定义导航栏、微信原生胶囊、页面主体或底部操作栏。

## 5. 埋点与非阻断

- [ ] AC-027 分享给微信朋友时 SHOULD 记录页面、渠道和关键业务 ID。
- [ ] AC-028 分享到朋友圈时 SHOULD 记录页面、渠道和关键业务 ID。
- [ ] AC-029 埋点失败不得阻断分享，不得弹出技术错误。
- [ ] AC-030 分享行为不得影响页面刷新、下拉刷新、加载更多、收藏、品牌跳转等既有交互。

## 6. 小程序平台与设备验收

- [ ] AC-031 不得在 WXML / WXSS 中自绘模拟微信分享、关闭或系统胶囊。
- [ ] AC-032 支持分享的页面必须继续使用微信小程序原生分享能力。
- [ ] AC-033 DevTools 320 / 375 / 430 pt 或等价静态视口验收应覆盖首页、商品详情页、商品列表页、品牌详情页的分享直达状态。
- [ ] AC-034 真机验收若无法执行，必须标记 `blocked` 或 `follow_up`，不得写作“真机通过”。
- [ ] AC-035 验收 evidence 必须覆盖微信原生胶囊避让、返回兜底、内容 offset 和分享路径关键参数。
- [ ] AC-036 小程序运行事实源必须验证实际运行 `.js` 与维护源码 `.ts` 的分享配置一致。

## 7. API / DB / 安全边界

- [ ] AC-037 默认不新增 API；如实现阶段新增或调整接口，必须同步 OpenAPI、Orval、docs 和测试。
- [ ] AC-038 默认不新增数据库表或字段；如新增分享配置持久化，必须补充 schema、数据库文档和测试。
- [ ] AC-039 不得放宽 MinIO 或媒体资源访问权限；分享图必须使用现有公开安全 URL 或本地兜底资源。
- [ ] AC-040 不得在分享路径、埋点或 evidence 中记录真实客户隐私、密钥、Cookie、Authorization header 或本机绝对路径。
