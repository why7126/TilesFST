# 一句话

微信小程序每次进入时，在右上角分享按钮附近展示“添加到我的小程序”的引导语，并允许用户手工关闭，方便用户下次快速找回小程序。

# 原始描述

每次进入小程序时，在右上角的分享按钮都有一个引导语，点击XXX按钮，添加到我的小程序，方便下次找回，允许用户手工关闭

# 背景与关联

- 涉及端：微信小程序
- 业务价值：提升用户对小程序收藏/找回入口的感知，降低下次访问路径成本
- 拆分判断：引导语展示、关闭行为和“添加到我的小程序”提示属于同一小程序留存引导交付闭环，记录为单条需求

# 待澄清

- [ ] “XXX按钮”具体指微信小程序右上角胶囊菜单中的哪个入口，还是页面内自定义分享/更多按钮
- [ ] 引导语是否每次进入都展示，还是用户关闭后在当前会话、当天或永久不再展示
- [ ] 引导语展示位置、样式、动效和遮挡规则是否需要适配自定义导航栏与不同机型安全区
- [ ] 是否需要埋点统计展示、关闭、点击分享/更多菜单后的转化情况
- [ ] 是否需要按页面范围展示：全局所有页面、首页优先，还是仅首次进入首页展示
