# 菲尚特瓷砖微信小程序：SKU 详情页原型上下文工程（v1）

## 1. 项目定位

面向装修客户、设计师、导购和品牌访客的数字瓷砖展厅。详情页用于展示、收藏和分享，不承担交易、库存、购物车或后台编辑。

## 2. 参考基线

本页面必须延续 `feishangte-miniapp-home-v6`：

```css
:root {
  --brand: #C8A055;
  --bg-primary: #18160F;
  --bg-secondary: #211E16;
  --text-primary: #EDE8DF;
  --text-secondary: rgba(237,232,223,0.5);
  --divider: rgba(255,255,255,0.07);
  --font-family: "Source Han Sans SC", "PingFang SC", sans-serif;
}
```

禁止改成暖白、电商红、亮绿色或纯黑体系。

## 3. 页面目标排序

1. 大图/视频呈现瓷砖纹理与质感。
2. 让用户快速确认名称、品牌、规格、工艺、色系和价格。
3. 收藏 SKU 作为选砖备选。
4. 分享给客户或家人。
5. 进入品牌主页并继续浏览同系列/同品牌商品。

## 4. 页面结构

```text
SkuDetailPage
├── StatusBar
├── NavigationBar
├── MediaGallery (390×430)
├── BrandSummaryCard
├── SkuSummaryCard
├── ProductParameterCard
├── RemarkCard
├── SameSeriesCarousel
├── SameBrandCarousel
└── BottomActionBar
```

## 5. 尺寸与视觉

- 原型基准宽度：390px。
- 页面左右边距：16px。
- 卡片圆角：16px；推荐卡片 12px。
- 卡片背景：`#211E16`。
- 细边框：`rgba(255,255,255,0.07)`。
- 主标题：22px / 700。
- SKU 名称：24px / 700。
- 卡片标题：16px / 700。
- 正文：13px。
- 辅助信息：11px。
- 价格：26px / 700，品牌金。
- 底部固定操作栏高度：84px + 安全区。

## 6. 媒体区

- 大图铺满顶部，不使用小卡片包裹。
- 图片和视频混合轮播，主图第一。
- 右下角显示“图片 6 / 视频 2”。
- 视频封面中央使用金色播放按钮。
- 图片使用高质量瓷砖纹理，避免通用电商白底商品图。
- 点击图片进入纯黑全屏预览。

## 7. 卡片组件

### BrandSummaryCard

Logo 40×40，品牌名 16px，副文案 11px，右侧箭头。

### SkuSummaryCard

品牌卡下方独立信息卡：SKU 名称、SKU 编码、参考价格。价格是视觉第二焦点，仅次于主图。

### ProductParameterCard

双列参数网格，类目路径独占一行。参数全部展开。

### RecommendationCard

宽 152px，图片高 132px；名称 14px，规格 11px，价格 13px。横向滑动并露出下一张。

## 8. 底部操作栏

- 左侧两个次级入口：收藏、品牌。
- 右侧分享为主按钮，占较大宽度。
- 未收藏使用线性心形；已收藏使用品牌金实心心形。
- 操作栏背景与首页 TabBar 一致，使用轻微毛玻璃和顶部分割线。

## 9. 原型交互

- 左右按钮和拖动模拟媒体切换。
- 图片点击打开全屏预览；双击模拟缩放。
- 视频项点击播放/暂停。
- 收藏按钮切换状态并 Toast。
- 品牌卡、品牌按钮、推荐卡模拟跳转 Toast。
- 分享按钮打开底部分享面板。
- URL 查询参数 `?state=preview|share|favorite` 用于截图不同状态。

## 10. 高保真生成提示词

生成 390px 宽的菲尚特瓷砖微信小程序 SKU 详情页。沿用首页 v6 深色企业轻奢风：品牌金 #C8A055、页面 #18160F、卡片 #211E16、主文字 #EDE8DF。顶部是大尺寸瓷砖纹理图片/视频混合轮播，占首屏约 70%，右下角显示图片和视频数量。下方突出品牌 Logo 与品牌入口，再展示 SKU 名称、编码、参考价格。商品参数卡完整展开品牌、多级类目、规格、表面工艺、主色系；备注独立卡片。底部有同系列和同品牌横向商品卡。固定底栏包含收藏、品牌和金色分享主按钮。整体克制、高端、真实可开发，不出现购物车、购买、下单、询价和夸张渐变。

## 11. 负面约束

- 不出现电商购物车、立即购买、销量、优惠券、促销倒计时。
- 不使用纯白大背景或红色主按钮。
- 不把所有参数挤进一行。
- 不在页内切换规格。
- 不隐藏多级类目。
- 不把备注折叠。
- 不用过度玻璃拟态、霓虹或复杂 3D 装饰。
