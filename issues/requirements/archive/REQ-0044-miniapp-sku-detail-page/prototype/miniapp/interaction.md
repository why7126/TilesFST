# 瓷砖 SKU 详情页交互说明

- 版本：v1.0
- 对应需求：REQ-0044

## 1. 页面进入

1. 页面接收 `skuId` 和可选 `source`。
2. 先渲染深色骨架屏，保持媒体区高度稳定，防止布局跳动。
3. 详情加载成功后渐显真实内容；失败则替换为错误状态。
4. 从分享进入时不改变页面能力，但记录来源埋点。

## 2. 顶部导航

- 左侧返回：优先返回上一页；无页面栈时进入首页。
- 右侧微信胶囊仅作为系统区域视觉占位，原生开发时使用系统胶囊定位。
- 页面向下滚动超过媒体区 60% 后，导航背景由透明过渡为 `#18160F`，标题显示 SKU 名称。

## 3. 图片与视频混合轮播

### 3.1 默认状态

- 首项为商品主图。
- 底部中央显示圆点/短条指示器，最多展示 7 个；媒体更多时使用当前序号。
- 右下角媒体统计显示“图片 N、视频 M”。
- 视频项中央显示播放按钮与时长。

### 3.2 手势

- 单指左右滑：切换上一项/下一项。
- 滑动距离超过容器宽度 20% 或速度超过阈值时完成切换，否则回弹。
- 图片项单击：进入全屏预览。
- 视频项单击播放按钮：原位播放；单击非按钮区域也可进入全屏播放。
- 用户开始手动滑动后，本次页面访问不再自动轮播。

### 3.3 视频

- 默认不自动播放、不抢占音频焦点。
- 点击播放后显示控制条。
- 播放时禁止轮播切换；用户主动滑动到其他项时暂停当前视频。
- 页面进入后台、锁屏、跳转其他页面时暂停。
- 多视频之间不允许同时播放。

## 4. 全屏图片预览

- 背景为纯黑，顶部显示返回和 `当前/总图片数`。
- 双击图片：在 1× 与 2.5× 之间切换，放大中心以点击位置为准。
- 双指缩放范围 1×–4×。
- 放大状态下单指拖动图片；图片边缘具有限位和轻微回弹。
- 1× 状态下左右滑切换图片。
- 长按图片调起微信图片菜单，支持保存和转发；需遵守微信权限规范。
- 切换图片后缩放比例重置为 1×。
- 点击“原图”加载高清资源，显示进度；失败可重试。

## 5. 品牌信息

- 品牌卡整卡可点击，底部“品牌”按钮与其行为一致。
- 按压反馈：背景提亮，整体缩放至 0.98，持续约 160ms。
- 点击后进入品牌主页，页面参数为 `brandId`。

## 6. SKU 摘要

- SKU 名称最多两行。
- 编码支持长按复制；复制成功提示“SKU 编码已复制”。
- 价格使用品牌金突出，单位字号低一级。
- “参考价格”始终展示，避免被理解为成交价。

## 7. 商品参数

- 默认全部展开，不提供折叠按钮。
- 参数采用双列键值布局；类目路径独占整行。
- 类目层级过长时允许换行，不横向滚动。
- 空值统一显示“—”，不隐藏字段，便于用户理解信息缺失。

## 8. 备注说明

- 独立卡片，完整展开。
- 支持换行和简单富文本强调，不支持外链脚本。
- 无备注时模块隐藏。
- 卡片底部固定弱提示：“参考价格及产品信息以门店最终确认为准”。

## 9. 推荐区

- “同系列商品”和“同品牌推荐”分别横向滑动。
- 卡片露出下一张约 20px，提示可继续滑动。
- 整卡点击进入新 SKU；收藏按钮单独响应，不触发跳转。
- 返回上一 SKU 时保留原页面滚动位置。

## 10. 底部固定操作栏

### 10.1 收藏

- 未收藏：线性心形图标 +“收藏”。
- 已收藏：金色实心心形 +“已收藏”。
- 点击后立即播放 180ms 缩放动画，再发起请求。
- 请求失败则回滚并提示“操作失败，请重试”。

### 10.2 品牌

- 使用品牌/门店线性图标。
- 点击进入品牌主页。

### 10.3 分享

- 使用主按钮样式，品牌金背景、深色文字。
- 点击调起微信分享面板。
- HTML 原型用底部分享弹层模拟。

## 11. Toast、Loading 与错误

- Toast 位于底部操作栏上方 20px，展示 1.2 秒。
- 请求按钮 Loading 不使用全屏遮罩。
- 详情加载失败提供“重新加载”和“返回首页”。
- 图片或视频单项失败不阻断其他内容。

## 12. 动效参数

| 动效 | 时长 | 曲线 |
|---|---:|---|
| 页面内容渐显 | 240ms | ease-out |
| 卡片按压 | 160ms | ease-out |
| 收藏心形缩放 | 180ms | cubic-bezier(.2,.8,.2,1) |
| 分享弹层进入 | 260ms | cubic-bezier(.2,.8,.2,1) |
| 全屏预览进入 | 220ms | ease-out |
| Toast | 180ms | ease-out |

## 13. 无障碍与可用性

- 所有图标按钮提供可读文本或 aria-label。
- 金色文字与深色背景保持足够对比度。
- 价格、名称等关键信息不只依靠颜色区分。
- 推荐卡片、品牌入口和底部按钮触控区域不小于 44px。
