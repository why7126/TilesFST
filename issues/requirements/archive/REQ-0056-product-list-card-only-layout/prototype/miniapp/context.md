# 原型上下文

## 目标

本原型用于表达微信小程序商品列表页的目标信息架构：去掉搜索、筛选和排序控件，将首屏空间让给双列商品卡片。视觉参考首页“热销推荐”，并延续深色石材风、品牌金价格强调和低圆角卡片。

## 页面结构

```text
custom-navigation
页面标题 / 入口上下文
双列商品卡片 grid
加载更多 / 无更多 / 空态 / 错误态
```

## 交互说明

- 商品卡片整卡点击进入 SKU 详情。
- 下拉刷新和上拉加载更多保留。
- 商品列表页不提供搜索框、筛选按钮、排序 tabs、筛选 chips 或筛选抽屉。
- 入口上下文只用于标题、初始查询和空态文案，不在页面内暴露二次筛选控件。

## 视觉验收

- 320 / 375 / 430 pt 视口下，每行 2 个卡片均不溢出。
- 长商品名限制为 2 行，品牌/规格和价格不遮挡。
- 缺图商品展示统一占位。
- 自定义导航与内容区、底部 TabBar 与列表底部留白不重叠。
