## US-001 后台运营维护瓷砖 SKU 主数据

**作为** 后台运营人员，  
**我希望** 在管理后台统一维护 SKU 名称、编码、品牌类目、规格工艺、参考价格、图片与视频素材，  
**以便** 商品资料完整、素材可追踪，并支持上下架管理。

### 验收要点

- 可访问瓷砖 SKU 管理页，Sidebar OPERATIONS 下「瓷砖SKU」高亮。
- 列表展示 SKU 信息（主图缩略图、名称、编码）、品牌/类目、规格/工艺、参考价格、素材完整度、状态、更新时间。
- 支持关键词（名称/编码）、品牌、类目、状态、素材完整度筛选；查询与重置可用。
- 展示 SKU 总数、已上架、待完善、草稿四个指标卡。
- 点击「新增SKU」打开弹窗；支持保存草稿与创建 SKU；新建默认草稿，弹窗无状态字段。
- 弹窗支持多图上传、设主图、多视频上传；价格字段为「参考价格（元）」。
- 可编辑 SKU、上下架/恢复；缺主图时允许草稿保存，列表素材列显示「缺主图」。

### 关联功能

- FR-001：SKU 列表与筛选
- FR-002：数据概览指标
- FR-003：新增/编辑 SKU 弹窗
- FR-004：多图与主图管理
- FR-005：多视频管理
- FR-006：参考价格展示
- FR-007：上下架 / 恢复

## US-002 后台管理员监管 SKU 与删除

**作为** 后台管理员，  
**我希望** 在符合业务规则时删除无用 SKU，并受权限体系约束，  
**以便** 主数据质量可控、已上架商品不被误删。

### 验收要点

- 已上架 SKU 不可删除；仅草稿/已停用且无业务引用时可删除。
- 删除需确认；服务端二次校验状态与引用。
- 管理员与运营均可维护 SKU（具体 RBAC 见 requirement.md §3）。

### 关联功能

- FR-008：条件删除
- FR-009：权限与 API 鉴权

---

## US-003 只读账号查看 SKU 资料

**作为** 只读账号，  
**我希望** 仅查看 SKU 列表与详情，不看到新增、编辑、删除入口，  
**以便** 内部协作查看资料而不误改数据。

### 验收要点

- 可进入列表页并浏览筛选结果。
- 不展示「新增SKU」、行内编辑、上下架、更多删除等写操作入口。

### 关联功能

- FR-009：权限与 API 鉴权

---

## US-004 前台展示依赖准确 SKU 数据（受益）

**作为** 店主或前台访客，  
**我希望** 仅看到已上架且素材完整的 SKU，  
**以便** 浏览体验准确、主图与视频可用。

### 验收要点

- 已上架 SKU 可在后续店主端/目录页消费（本期实现管理端，前台为下游）。
- 停用/草稿 SKU 不在前台展示。

### 关联功能

- FR-007：上下架（下游消费）
