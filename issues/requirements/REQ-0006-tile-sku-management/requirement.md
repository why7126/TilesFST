# 瓷砖SKU管理页面需求文档

## 1. 背景与目标

瓷砖 SKU 是 TilesFST 管理后台最核心的商品主数据。本页面用于后台运营维护 SKU 基础资料、规格参数、价格、图片素材、视频素材与发布状态，并提供「新增SKU」录入能力。

v4 在 v3 基础上重点优化 SKU 新增/编辑弹窗：SKU 支持多张图片且必须可指定一张主图；支持上传多个视频；价格字段明确展示单位「参考价格（元）」；新增/编辑弹窗不再出现状态字段，新建 SKU 保存后默认进入「草稿」状态。

## 2. 设计约束

- 必须继承管理后台首页的暗色旗舰风、固定左侧导航、右侧独立滚动布局。
- Sidebar 宽度为 264px，当前导航高亮「瓷砖SKU」。
- 页面背景、卡片、表格、按钮、状态标签沿用 ui-design.md 视觉 Token：暗色背景、品牌金强调、0.5px 细线、2-3px 低圆角。
- 主操作「新增SKU」使用品牌金实底按钮，位置在页面标题区右侧。
- HTML、原型图、原型上下文必须拆分成不同文件；列表态与新增SKU弹窗态分别交付。

## 3. 用户角色与权限

| 角色 | 权限 |
|---|---|
| 后台运营 | 查看、筛选、新增、编辑、上下架 SKU，维护图片与视频 |
| 后台管理员 | 查看、筛选、新增、编辑、上下架、删除符合条件的 SKU |
| 只读账号 | 仅查看列表与详情，不展示新增、编辑、删除入口 |

## 4. 页面信息架构

### 4.1 SKU列表页

页面从上到下包含：

1. 左侧固定 Sidebar。
2. 页面标题区：标题、说明、主操作「新增SKU」。
3. 数据概览：SKU总数、已上架、待完善、草稿。
4. 筛选区：关键词、品牌、类目、状态、素材完整度。
5. SKU列表表格。
6. 分页区：左侧显示 `共 x 条`；右侧显示页码选择与每页条数。

### 4.2 新增/编辑SKU弹窗

弹窗字段从上到下：

1. SKU名称（必填）
2. SKU编码（必填）
3. 所属品牌（必填）
4. 所属类目（必填）
5. 规格尺寸（必填）
6. 表面工艺（选填）
7. 主色系（选填）
8. 参考价格（元）（必填，新建默认 0）
9. SKU图片（支持多张，必须允许设置其中一张为主图）
10. SKU视频（支持多个视频）
11. 备注说明（选填）

弹窗不展示「状态」字段。新增 SKU 默认状态为草稿。

## 5. 功能需求

### 5.1 SKU列表

- 默认按更新时间倒序展示。
- 列表字段：SKU信息、品牌/类目、规格/工艺、参考价格、素材、状态、更新时间、操作。
- SKU信息展示 SKU 名称、SKU 编码和主图缩略图。
- 素材列展示图片数量、视频数量和主图是否已设置。
- 状态包括：已上架、草稿、待完善、已停用。
- 操作包括：编辑、上下架/恢复、更多。

### 5.2 筛选与搜索

| 筛选项 | 类型 | 说明 |
|---|---|---|
| 关键词 | 输入框 | 支持 SKU名称、SKU编码模糊搜索 |
| 品牌 | 下拉 | 全部品牌 / 指定品牌 |
| 类目 | 下拉 | 全部类目 / 指定类目 |
| 状态 | 下拉 | 全部状态 / 已上架 / 草稿 / 待完善 / 已停用 |
| 素材完整度 | 下拉 | 全部 / 已完整 / 缺主图 / 缺图片 / 缺视频 |

点击「查询」刷新列表；点击「重置」清空筛选条件并恢复默认列表。

### 5.3 新增SKU

- 点击「新增SKU」打开新增SKU弹窗。
- 弹窗支持「保存草稿」与「创建SKU」。
- 必填项校验失败时，字段边框变为风险色，并在字段下方展示错误文案。
- 创建成功后关闭弹窗，列表刷新至第一页，并展示成功提示。
- 新建 SKU 默认状态为「草稿」，前端不提供状态选择控件。

### 5.4 图片管理

- SKU 允许上传多张图片。
- 每个 SKU 必须支持指定一张主图。
- 上传图片后以缩略图网格展示，主图显示「主图」标记。
- 支持切换主图、删除非主图图片、继续添加图片。
- 若未设置主图，保存时允许保存草稿，但列表素材完整度显示「缺主图」。

### 5.5 视频管理

- SKU 允许上传多个视频。
- 视频以文件卡片展示，包含视频名称、时长/大小、上传状态与删除入口。
- 支持继续添加视频。
- 视频不是创建 SKU 的必填项。

### 5.6 价格字段

- 字段名称必须展示为「参考价格（元）」。
- 输入值为数字，支持两位小数。
- 列表显示时格式化为 `¥ 268.00`。

### 5.7 分页

- 分页区固定在列表卡片底部。
- 左侧显示：`共 12,860 条`。
- 右侧显示：页码选择 + 每页条数。
- 页码选择包含上一页、页码、下一页；当前页使用品牌金实底。
- 每页条数支持 10 / 20 / 50 / 100 条，默认 20 条。

## 6. 状态与校验

| 场景 | 处理 |
|---|---|
| 无数据 | 表格区域展示空状态、说明文案和「新增SKU」入口 |
| 加载中 | 表格行展示 skeleton，不改变筛选区布局 |
| 创建成功 | toast：SKU创建成功，已保存为草稿 |
| 创建失败 | 保留弹窗输入，展示服务端错误 |
| 缺主图 | 允许草稿保存，列表素材列显示缺主图 |
| 删除限制 | 已上架 SKU 不允许删除；仅草稿/已停用且无业务引用时允许删除 |

- 当前导航高亮为「瓷砖SKU」。
- 列表页存在「新增SKU」按钮。
- 新增SKU弹窗为独立 HTML 与独立原型图。
- 新增SKU弹窗支持多图上传、设置主图、多个视频上传。
