## US-001 后台管理员更高效地筛选与浏览用户列表

**作为** 后台管理员，  
**我希望** 用户列表页的搜索与分页信息更简洁、用户名列更易读，  
**以便** 减少无效点击、快速定位目标用户。

### 验收要点

- 筛选区无「搜索」按钮；修改关键词或筛选项后列表自动刷新；「重置」可清空条件。
- 搜索框提示「搜索用户名/昵称」，且搜索结果不包含邮箱/手机号匹配。
- 列表区无「用户列表」标题行与「仅后台管理员可编辑用户」提示行。
- 用户列用户名与昵称分两行显示；无昵称时第二行显示「未设置昵称」。
- 分页左侧显示「共 x 个用户」，右侧为页码与「每页显示 x 条」，无「1-10 / N」等冗余文案。
