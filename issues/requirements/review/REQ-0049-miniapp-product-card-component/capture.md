# 微信小程序商品卡片组件

生成微信小程序商品卡片组件，应用于各类商品列表页的展示与交互，统一商品核心信息呈现、图片展示、状态提示、点击跳转和列表场景复用方式。

# 原始描述

生成微信小程序的商品卡片组件，应用于 各类商品列表页的展示与交互

# 待澄清

- [ ] 商品卡片需要展示哪些字段：商品图、名称、品牌、型号、规格、价格、标签、库存/上下架状态等。
- [ ] 卡片交互范围：点击进入 SKU 详情、收藏、分享、加入询价/对比、图片预览，或仅支持详情跳转。
- [ ] 需要应用到哪些小程序列表页：分类列表、搜索结果、首页推荐、收藏列表或后续商品聚合页。
- [ ] 卡片视觉是否需要适配现有小程序首页、分类页、商品列表页和自定义导航栏风格。
- [ ] 是否需要空态、加载态、骨架屏或埋点事件随卡片组件一起沉淀。
