## US-001 小屏查看管理端列表

**作为** 后台管理员 / 内部员工，  
**我希望** 在手机或小屏平板上访问当前已实现的管理端列表页时，筛选区、表格、分页和行内操作不会重叠或整体横向溢出，  
**以便** 临时查看、筛选和执行单项管理操作时不被布局问题阻断。

### 验收要点

- 覆盖 `/admin/brands`、`/admin/banners`、`/admin/tile-categories`、`/admin/tile-specs`、`/admin/tile-skus`、`/admin/users`、`/admin/logs`、`/admin/api-docs`。
- `375x812`、`390x844`、`768x1024` 视口下页面级无不可控横向滚动。
- 筛选区可单列或两列降级，查询、重置按钮可点击。
- 宽表格仅在表格容器内横向滚动；操作列或关键操作可访问。
- 分页控件可换行或分组展示，不压缩到不可点击。

### 关联功能

- FR-001、FR-002、FR-003、FR-004、FR-008

## US-002 小屏操作管理端弹窗

**作为** 后台运营人员，  
**我希望** 在手机或窄屏窗口打开新增/编辑/确认类弹窗时，弹窗内容可滚动、关闭入口和底部主操作可访问，  
**以便** 不因弹窗高度或宽度问题中断品牌、Banner、SKU、用户等维护操作。

### 验收要点

- 新增/编辑弹窗在 375px 宽度下不超出视口；内容区可滚动。
- SKU、Banner 等宽弹窗保留专属宽度策略，窄视口下 `max-width` 生效。
- 删除、启停、上下架、重置密码等确认弹窗使用 DS modal；无 `window.confirm`。
- 错误提示出现后不导致按钮错位或底部操作丢失。
- 含图片/视频/头像/Logo 上传的弹窗不破坏已有上传状态机和同会话回显。

### 关联功能

- FR-005、FR-008

## US-003 小屏维护设置与个人资料

**作为** 后台管理员，  
**我希望** 系统设置、个人资料、修改密码等表单型页面在移动视口下保持单列可读，保存/重置/取消等操作明确且不重复，  
**以便** 在移动设备上临时完成配置查看或少量维护。

### 验收要点

- `/admin/settings/:tab` 设置导航、表单字段和保存/重置操作可访问。
- `/admin/profile` 主信息与账号安全区域在小屏下单列可读。
- 全页保存 CTA 不重复；保存成功反馈使用 fixed toast，不推挤布局。
- 恢复默认、dirty Tab 切换等风险操作使用 DS modal。

### 关联功能

- FR-006、FR-008

## US-004 固定视口完成移动端验收

**作为** QA / 验收人员，  
**我希望** 有明确的移动端视口、页面清单和检查项，  
**以便** 后续 apply 阶段可以稳定判断 Web 管理端移动端基础适配是否达标。

### 验收要点

- 固定验收视口：`375x812`、`390x844`、`768x1024`、`1440x1024`。
- 必测路由：`/admin/login`、`/admin/dashboard`、`/admin/tile-skus`、`/admin/brands`、`/admin/users`、`/admin/logs`、`/admin/settings/basic`。
- 每个必测路由记录页面级横向溢出、控件重叠、弹窗不可关闭、底部按钮不可达、筛选/分页不可操作等结果。
- 横切 AC（knowledge-base）全部有对应验收记录或 N/A 理由。

### 关联功能

- FR-008

## US-005 非管理端用户不受影响（边界）

**作为** 店主 Web 用户或微信小程序用户，  
**我希望** 本次管理端移动端适配不改变我的访问页面和展示逻辑，  
**以便** 店主端与小程序能力不被管理端样式变更误伤。

### 验收要点

- 本 REQ 不修改店主 Web 展示端、小程序页面、后端 API、数据库或 Orval。
- 若后续 change 实现阶段触及上述范围，必须退回 PRD 或拆分独立 REQ。

### 关联功能

- FR-001
