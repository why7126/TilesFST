# REQ-0007 启停确认弹窗 — 原型上下文

## 1. 参考

- 删除确认：已实现 `TileCategoryManagementPage` 删除弹窗（标题「删除类目」）
- 本弹窗 MUST 复用相同 modal 样式类

## 2. 停用

| 元素 | 内容 |
|------|------|
| 标题 | 停用类目 |
| 正文 | 确认停用类目「{name}」？停用后前台将不再展示该类目。 |
| 取消 | 取消 |
| 确认 | 确认停用（主按钮） |

## 3. 启用

| 元素 | 内容 |
|------|------|
| 标题 | 启用类目 |
| 正文 | 确认启用类目「{name}」？ |
| 取消 | 取消 |
| 确认 | 确认启用（主按钮） |

## 4. 交互

- 打开弹窗时不调用 API
- 确认后调用 `POST .../disable` 或 `POST .../enable`
- ESC / 遮罩 / × 等同「取消」
