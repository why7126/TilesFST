## US-001 按需收起侧栏以扩大主内容区

**作为** 后台管理员，  
**我希望** 在桌面端点击侧边栏头部右上角的 chevron 将导航栏收成窄图标列，  
**以便** 浏览宽表格或表单时有更多横向空间。

### 验收要点

- 视口 > 1023px 时，侧栏由 264px 平滑过渡到 72px。
- chevron 位于头部右上角；展开态指向左，收起态指向右。
- 主内容区随侧栏变宽自动扩展，无横向滚动条。
- 过渡动画约 200–250ms，尊重 `prefers-reduced-motion`。

### 关联功能

- FR-002、FR-003、FR-005（见 `requirement.md`）

## US-002 收起后仍能导航与识别当前页

**作为** 后台管理员，  
**我希望** 侧栏收起后仍能通过图标跳转各菜单，并清楚看到当前所在页面，  
**以便** 在窄侧栏模式下不影响日常操作。

### 验收要点

- 收起态隐藏导航文案与分区标题，保留 `.nav-icon`。
- 当前路由对应项保留 `active` 样式（金色 + 左侧 accent）。
- 点击 nav 项行为与展开态一致（路由跳转 / placeholder 提示）。
- 每项具备 `aria-label` 或等价读屏名称。

### 关联功能

- FR-005、FR-006（见 `requirement.md`）

---

## US-003 记住我的侧栏偏好

**作为** 后台管理员，  
**我希望** 系统记住我上次选择的展开或收起状态，  
**以便** 切换 `/admin/*` 页面或下次登录后无需重复设置。

### 验收要点

- 首次访问默认 **expanded**。
- 切换后写入 `localStorage`（`admin-sidebar-collapsed`）。
- 刷新页面或导航至其他管理页后状态保持一致。
- 收起态底部用户区仅显示头像，点击仍可打开个人资料 / 退出等菜单。

### 关联功能

- FR-001、FR-005、FR-006（见 `requirement.md`）
