## Acceptance

### AC-001 品牌列表分页与用户管理页一致

- [ ] `/admin/brands` 与 `/admin/users` 底部分页区域的布局、按钮尺寸、边框、圆角、字号、激活态和每页显示控件一致。
- [ ] 品牌页不再出现割裂的 `page-left` / `brand-pagination-right` 主视觉结构。

### AC-002 品牌分页功能不回退

- [ ] 上一页、下一页、当前页展示可用。
- [ ] 每页显示切换可用，切换后页码重置为 1。
- [ ] 若保留跳页输入，其视觉被纳入统一分页布局。

### AC-003 品牌 Logo 文件选择控件一致

- [ ] 新增/编辑品牌弹窗中的 Logo 控件与用户管理弹窗头像上传控件视觉层级一致。
- [ ] 文件选择入口使用明确按钮或可访问 label。
- [ ] 原生 file input 不展示默认浏览器控件皮相。
- [ ] Logo 空态、预览态、帮助文案不挤压弹窗布局。

### AC-004 品牌管理功能不回退

- [ ] 查询、重置、分页、新增品牌、编辑品牌、上传 Logo、保存品牌可用。
- [ ] 不修改 API 请求/响应、数据库字段、权限边界和 MinIO 策略。

### AC-005 测试与构建

- [ ] 品牌管理相关 Vitest 通过。
- [ ] Web 构建通过。
- [ ] 验收结论写入 `trace.md`。
