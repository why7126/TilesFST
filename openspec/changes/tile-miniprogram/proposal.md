## Why

瓷砖销售公司的渠道商（零售店老板）需要通过移动端便捷地浏览瓷砖产品信息、查看产品详情、收藏感兴趣的产品。目前缺乏移动端入口，渠道商只能通过线下或电脑端查看产品，效率较低。小程序是最适合中国渠道商的使用习惯，无需安装、即用即走，能够满足渠道商随时随地浏览产品的需求。

## What Changes

- 新增微信小程序端产品展示系统
- 实现产品列表展示，支持按品牌、尺寸、类目多维筛选和关键词搜索
- 实现产品详情页，支持图片轮播、视频播放、属性展示、价格展示
- 实现本地收藏功能，渠道商可收藏感兴趣的产品
- 实现本地浏览记录功能，记录用户最近浏览的产品
- 实现产品分享功能，支持将产品分享给终端客户

## Capabilities

### New Capabilities

- **product-list**: 产品列表展示，支持多维筛选和搜索
- **product-detail**: 产品详情展示，支持图片轮播、视频、属性、价格
- **product-favorite**: 本地收藏功能，收藏感兴趣的产品
- **product-history**: 本地浏览记录，记录浏览历史
- **product-share**: 产品分享功能

### Modified Capabilities

- 无

## Impact

- 新增小程序前端项目，使用 TypeScript 开发
- 依赖后端 API 提供产品数据（tile-api-backend）
- 数据存储在本地，不涉及服务端数据库
- 使用微信小程序开发框架
