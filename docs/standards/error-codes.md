# 错误码体系

## 分段规则

| 区间 | 含义 |
|------|------|
| `0` | 成功 |
| `1xxxx` | 系统错误 |
| `2xxxx` | 认证授权 |
| `3xxxx` | 业务错误 |
| `4xxxx` | 参数错误 |
| `5xxxx` | 外部依赖（MinIO、存储等） |

## 已登记错误码

| 码 | HTTP | 说明 |
|----|------|------|
| 0 | 200 | 成功 |
| 10001 | 500 | 系统内部错误 |
| 20001 | 401 | 未登录或 token 无效 |
| 20002 | 401 | 登录凭证错误 |
| 20003 | 403 | 账号已停用 |
| 20004 | 403 | 无权限 |
| 30001 | 404 | 瓷砖不存在 |
| 30002 | 409 | 瓷砖名称冲突 |
| 30031 | 409 | SKU 编码重复 |
| 30032 | 409 | SKU 删除禁止（已上架等） |
| 30033 | 409 | SKU 上架禁止（资料不完整等） |
| 30040 | 404 | 瓷砖规格不存在 |
| 30041 | 409 | 瓷砖规格尺寸重复 |
| 30042 | 409 | 瓷砖规格删除禁止（仍有关联 SKU 或未停用） |
| 30043 | 409 | 瓷砖规格已停用（不可绑定新 SKU） |
| 30050 | 404 | Banner 不存在 |
| 30051 | 409 | Banner 标题重复（同展示端+位置） |
| 30052 | 400 | Banner 跳转目标无效 |
| 30053 | 409 | Banner 删除禁止（已上线） |
| 30054 | 400 | Banner 外部链接无效（须 https://） |
| 40001 | 400 | 请求参数无效 |
| 40002 | 400 | 分页参数无效 |
| 40020 | 400 | 原密码错误（自助改密） |
| 40021 | 400 | 新密码不符合策略（8–32 位、含字母与数字） |
| 40022 | 400 | 弱密码 |
| 40023 | 400 | 新密码不能与原密码相同 |
| 42901 | 429 | 改密操作过于频繁 |
| 50001 | 502 | 对象存储不可用 |
| 50002 | 400 | 文件类型不允许 |
| 50003 | 400 | 文件大小超限 |

## 实现

- Python 常量：`src/backend/app/core/error_codes.py`
- 异常：`src/backend/app/core/exceptions.py`

## AI 规则

新增业务错误 MUST 在本表与 `error_codes.py` 同步登记，禁止随意使用未文档化数字。
