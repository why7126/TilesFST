# 产品总览


## 产品目标

为瓷砖零售店店主提供统一的瓷砖信息查询入口与移动端展示橱窗，为企业内部员工提供瓷砖资料、Banner 和媒体素材维护后台。

## 核心场景

- 店主查看瓷砖图片、规格、分类、基础信息。
- 店主通过微信小程序首页浏览门店 Banner、新品推荐、热门推荐，并基于 SKU 主图和价格快速判断咨询意向。
- 企业员工新增、编辑、下架瓷砖产品。
- 企业员工维护分类、规格、产品图片、价格和面向小程序展示的 Banner；瓷砖类目统一按一级/二级两级结构维护，保持与微信小程序分类页交互一致。
