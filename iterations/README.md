# iterations

Sprint 迭代管理。生命周期阶段目录见 `rules/iterations-lifecycle.md`。

| 目录 | 含义 |
|---|---|
| `change/` | 未归档：规划中或开发中（`planning` / `in_progress`） |
| `archive/` | 已完成归档（`completed`） |

**新建**：`/sprint-propose` → `change/sprint-xxx/`（四件套）  
**迭代归档**：`/sprint-archive` → 迁入 `archive/`

遗留扁平 `iterations/sprint-xxx/` 仍可读，勿再新建。
