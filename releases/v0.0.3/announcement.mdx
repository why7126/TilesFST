# 产品版本 v0.0.3

发布时间：2026-07-15 13:48:54

关联 Sprint：sprint-005、sprint-006、sprint-007

## 新增功能

- 新增品牌证书管理能力：管理端支持维护品牌资质、检测报告、荣誉证书和前台展示状态，并支持证书文件上传、预览、显示、隐藏和删除。
- 新增账号级主题偏好：管理端支持主题舒适度优化与主题选择器侧边栏入口，主题偏好可随账号保存。
- 新增管理端列表页基础能力：沉淀 AdminListPage 契约、MetricCard、MetricCardGrid 和分页窗口工具，减少列表页体验漂移。
- 新增 Clipboard 复制 helper 与最佳实践文档，用于统一复制成功、失败和浏览器能力不足时的反馈。
- 新增 AI 使用量观测与工作流命令后置事实源能力，Sprint close / exps 可区分 actual、estimated fallback 与 unavailable。

## 修复 BUG

- 修复 Sprint 归档在存在未完成 tasks 时未硬阻断的问题。
- 修复 API governance route tags 历史债，降低 Swagger 与接口治理噪声。
- 修复 workflow-sync check 的时间漂移和幂等性问题。
- 修复创建用户、重置密码后的初始密码复制失败问题。
- 修复日志审计页复制 request_id 可能失败的问题。
- 修复修改密码策略失败时提示过于泛化的问题。
- 修复 issue 子文档归档状态残留和 archived Change trace 兜底摘要门禁问题。
- 修复主题选择器未位于侧边栏用户头像上方的问题。

## 发布注意事项

- 本版本包含 API、数据库、Web 管理端、对象存储上传和 Orval 同步范围，发布前必须使用目标环境执行 smoke。
- 品牌证书文件仍必须经后端鉴权和对象存储适配层上传，前端不得直连未授权对象存储。
- API 校验错误统一 envelope 后，前端错误展示应优先读取统一 message 和字段错误列表。
- AI 使用量事实源不得包含原始 prompt、系统指令、真实路径、密钥或工具输出全文。

## 已知问题

- 部分 archived Change 的历史 trace Frontmatter 可能保留旧状态文案；发布事实以归档目录、Issue trace、Sprint 验收报告和 release validation 为准。
- 生产环境仍需单独验证品牌证书上传大小、文件类型、媒体回显、品牌删除约束和权限边界。

## 升级步骤

1. 升级前备份数据库、对象存储数据与当前部署配置。
2. 确认用户可见产品版本为 v0.0.3。
3. 同步生产环境变量与部署配置，确认数据库、MinIO 单桶、媒体前缀、上传大小限制和 Web 代理配置。
4. 执行 SQLite/MySQL schema 初始化或迁移，确认主题偏好字段与品牌证书表结构已生效。
5. 重新生成 OpenAPI/Orval 客户端并构建 Web、Backend 镜像。
6. 启动服务后验证登录、主题切换、用户密码复制、日志 request_id 复制、API 校验错误展示、品牌证书 CRUD、证书文件上传和媒体回显。
7. 完成发布对象校验与公告静态检查后进入发布确认。

## 回滚说明

1. 停止当前版本服务并回退到上一 Git tag 或上一部署包。
2. 恢复上一版本环境变量、Docker Compose 配置和 Web/Backend 镜像。
3. 恢复升级前数据库备份；如已执行品牌证书或主题偏好相关迁移，按数据库回滚说明处理。
4. 恢复对象存储备份或回退 MinIO/S3 配置。
5. 重新执行登录、用户管理、主题偏好、日志审计、品牌管理、媒体读取和管理端首页 smoke 验证。

## 影响范围

| 范围 | 影响 |
|---|---|
| Web 管理端 | 新增品牌证书页、主题偏好、列表基础组件、复制 helper、错误 envelope 展示和多处交互修复 |
| 店主 Web | 无主功能交付；共享主题策略和后端能力上线后需基础 smoke |
| 小程序 | 无 |
| 后端 | 新增或完善主题偏好、品牌证书、统一校验错误、上传鉴权与工作流治理能力 |
| 数据库 | 新增主题偏好字段与品牌证书表、索引、约束和迁移 |
| 对象存储 | 品牌证书文件走 MinIO 单桶前缀策略与后端适配层 |
| Docker | Compose 拓扑不变；需验证 Web、Backend、MinIO 组合下的上传和媒体回显 |
